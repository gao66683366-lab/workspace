# 04-算法与数据处理

> **定位说明：** 算法与数据处理是智能综合检测的智慧核心——将传感器原始数据转化为结构化检测结论。本目录涵盖AI缺陷检测模型、几何参数计算算法、多维数据融合分析、训练数据集以及数据分析工具，是综合检测从"采集数据"到"输出结论"的技术支撑。

---

## 一、算法体系总览

综合检测的算法体系分为三层：

```
┌─────────────────────────────────────────────────────────────┐
│                      算法体系三层架构                         │
│                                                             │
│  第一层：感知算法（数据获取层）                                │
│  ─ 图像预处理 / 点云滤波 / 姿态解算 / 同步对齐                  │
│                                                             │
│  第二层：分析算法（认知判断层）                                │
│  ─ AI缺陷检测 / 廓形拟合 / 参数计算 / 判级分类                  │
│                                                             │
│  第三层：融合算法（综合决策层）                                │
│  ─ 多维数据融合 / 热点分析 / 趋势预测 / 健康评估                 │
└─────────────────────────────────────────────────────────────┘
```

| 层级 | 核心技术 | 输入 | 输出 |
|-----|---------|------|------|
| 感知算法层 | OpenCV / PCL / IMU滤波 | 原始相机图像/激光点云/陀螺仪数据 | 预处理后的干净数据 |
| 分析算法层 | YOLO / ResNet / 廓形匹配 | 预处理后数据 | 缺陷类型/位置/几何参数 |
| 融合算法层 | 多维特征融合 / 时序分析 | 各路分析结果 | 综合判级/趋势/健康画像 |

## 二、感知算法层

### 2.1 图像预处理算法

| 算法 | 作用 | 参数 |
|-----|------|------|
| 畸变校正 | 修正镜头畸变对测量的影响 | 内参标定系数 |
| 亮度均衡 | 消除光源波动影响，统一图像亮度分布 | 直方图均衡化 |
| ROI裁切 | 减少无效区域，降低计算量 | 按相机分区域配置 |
| 噪声滤波 | 去除随机噪声 | 高斯滤波/中值滤波 |
| 条纹抑制 | 消除激光光条过曝/反射干扰 | 多帧平均/自适应滤波 |

### 2.2 3D点云处理算法

| 算法 | 作用 |
|-----|------|
| 三角测量重建 | 根据激光光条形变还原轨头三维坐标 |
| 点云滤波（PCL） | 去除离群点，保留有效廓形点 |
| 廓形平滑 | 沿轨道方向多点云平均，减小随机误差 |
| 坐标系变换 | 将点云从传感器坐标系变换到轨道坐标系 |

### 2.3 姿态数据处理算法

| 算法 | 作用 |
|-----|------|
| 陀螺仪数据滤波 | 低通滤波消除高频振动噪声 |
| 姿态角解算 | 从原始角速度积分得到水平/高低/轨向角 |
| 零偏校准 | 上电时自动校准陀螺仪零偏，消除温漂影响 |
| 异常值剔除 | 剔除突变数据点（车辆通过接头时的冲击） |

## 三、分析算法层

### 3.1 AI缺陷检测算法

本系统的核心检测能力，基于深度学习的目标检测网络。

**模型架构：**

| 模块 | 选型 | 说明 |
|-----|------|------|
| 主干网络 | CSPDarknet / ResNet50 | 特征提取 |
| 颈部网络 | FPN + PAN | 多尺度特征融合 |
| 检测头 | YOLOv8检测头 | 高精度目标检测 |
| 分割分支 | UNet | 像素级缺陷区域分割（可选） |

**可检测缺陷类型（可扩展）：**

| 缺陷类别 | 检测方式 | 备注 |
|---------|---------|------|
| 轨面裂纹 | 目标检测 + 分割 | 细裂纹专项优化 |
| 轨面掉块 | 目标检测 | 边缘不规则区域 |
| 轨面波磨 | 目标检测 + 频率分析 | 周期性纹理异常 |
| 轨面凹陷 | 目标检测 + 3D数据 | 深度信息辅助 |
| 轨面腐蚀 | 目标检测 | 锈蚀特征提取 |
| 螺栓缺失 | 目标检测 | 目标缺失专项 |
| 螺栓松动 | 目标检测 + 位置偏差 | 相对位置分析 |
| 焊缝缺陷 | 目标检测 | 焊缝区域异常 |

**模型部署：**

| 项目 | 配置 |
|-----|------|
| 推理框架 | ONNX Runtime（支持边缘部署） |
| GPU加速 | CUDA + TensorRT INT8量化 |
| 输入尺寸 | 640×640（适配推理效率） |
| 推理速度 | ≥25 FPS（满足实时要求） |
| mAP指标 | 待训练后实测（目标≥0.85） |

### 3.2 几何参数计算算法

**3.2.1 轨头廓形计算**

```
输入：3D线激光点云（帧）→ 廓形点序列 → 标准廓形模板匹配 → 计算磨耗量
```

| 参数 | 计算方法 |
|-----|---------|
| 磨耗量 | 实测廓形与标准廓形模板的最大偏差 |
| 横移量 | 轨头中心相对轨道基准线的偏移 |
| 廓形偏差 | 各点实测值与标准值的偏差序列 |

**3.2.2 轨距计算**

```
输入：左右轨3D激光点云 → 轨顶面中心点 → 计算水平距离 → 轨距值
```

**3.2.3 姿态参数计算**

```
输入：陀螺仪角速度 → 积分 → 姿态角 → 滤波 → 水平/高低/轨向值
```

### 3.3 自动判级算法

对接《铁路线路检测标准规范》，将检测结果映射为等级。

**判级依据（需补充具体限值数据）：**

| 等级 | 典型条件（示例） |
|-----|---------------|
| 正常 | 偏差在允许范围之内 |
| 一级 | 偏差达到允许范围的60%~80% |
| 二级 | 偏差超过允许范围，需计划养护 |
| 三级 | 偏差严重超限，立即处置 |

> **注：** 具体限值需依据铁路局标准规范，本系统支持按标准文件灵活配置阈值。

## 四、融合算法层（综合决策核心）

### 4.1 多维数据融合框架

综合检测的最大优势在于多维数据联合分析：

```
┌──────────────────────────────────────────────────────────────┐
│                  多维数据融合分析框架                          │
│                                                              │
│   2D图像缺陷 ──┐                                              │
│                ├──▶ 缺陷位置关联 ──▶ 综合判级置信度提升        │
│   3D廓形数据 ──┤                                              │
│                │                                              │
│   姿态几何数据 ─┤                                              │
│                │                                              │
│   历史基线数据 ─┘                                              │
│                                                              │
│   结果：单点缺陷 ──▶ 附有姿态/廓形/趋势标签的综合结论          │
└──────────────────────────────────────────────────────────────┘
```

**融合逻辑示例：**

| 情况 | 2D图像 | 3D廓形 | 姿态 | 综合结论 |
|-----|--------|--------|------|---------|
| 案例A | 疑似裂纹 | 无变形 | 无异常 | 降级判为"疑似"，需人工复核 |
| 案例B | 明显裂纹 | 局部凹陷 | 同位置轨向偏 | 升一级判为严重，加强关注 |
| 案例C | 正常 | 磨耗超标 | 无异常 | 直接输出二级报警 |

### 4.2 缺陷热点分析算法

| 算法 | 作用 | 输入 |
|-----|------|------|
| 空间聚类（DBSCAN） | 将沿线缺陷按里程聚类，识别高发区段 | 缺陷里程+类型+等级 |
| 密度分析 | 计算各区段单位里程内缺陷数量 | 缺陷空间密度 |
| 趋势计算 | 同一点位历史数据的变化趋势 | 多期检测数据时序 |

### 4.3 预测性维护算法（规划阶段）

| 模型 | 说明 | 状态 |
|-----|------|------|
| 劣化趋势预测 | 基于历史数据预测缺陷扩展速度 | 规划中 |
| 剩余寿命预估 | 估算各区段到下次大修的剩余时间 | 规划中 |
| 养护优化模型 | 多约束条件下养护资源最优分配 | 规划中 |

> **注：** 大数据预测模型依赖长期数据积累，当前阶段重点是完善数据采集和基础分析能力。
