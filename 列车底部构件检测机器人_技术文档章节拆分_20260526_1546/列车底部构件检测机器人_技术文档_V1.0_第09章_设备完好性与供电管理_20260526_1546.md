# 第9章 设备完好性与供电管理

## 9.1 设备完好性监控体系

本系统建立覆盖全节点的设备健康监控体系，实时监测关键部件状态，提前预警，防止带病运行。

### 9.1.1 监控节点矩阵

| 监控对象 | 监控参数 | 阈值 | 处理策略 |
|---|---|---|---|
| 主控计算机 | CPU温度、内存占用 | CPU>85°C | 降频+报警 |
| AI加速单元 | GPU温度、显存占用 | GPU>80°C | 降频+报警 |
| 电池BMS | 电芯电压、温度、电流 | 单芯>3.65V或<2.5V | 报警+强制停机 |
| 电池BMS管理板 | 板载温度 | >60°C | 报警+降载 |
| 云台电机 | 电流、温度 | 电流>5A或温度>70°C | 报警+停止该方向 |
| 行走电机 | 电流、转速异常 | 电流>10A | 急停+报警 |
| 相机 | 帧率异常、连接中断 | 丢帧率>5% | 报警+切换备路 |
| 激光雷达 | 扫描频率、点云密度 | 密度<80%正常值 | 报警+减速 |
| 倾角传感器 | 平台倾斜角度 | 横滚>15°或俯仰>20° | 自动停机 |
| 通信链路 | 光纤中断、5G信号强度 | 信号<-100dBm | 自动切换链路 |

### 9.1.2 三级故障告警机制

| 级别 | 告警类型 | 触发条件 | 处理方式 |
|---|---|---|---|
| 一级（红色） | 紧急告警 | 碰撞信号、急停触发、倾角超限、严重过热 | 立即停机+声光报警+上传远程 |
| 二级（黄色） | 警告告警 | 电池电量<20%、通信链路降级、传感器数据异常 | 界面提示+继续作业+安排收工 |
| 三级（蓝色） | 提示告警 | 温度偏高、轻微振动、存储容量>80% | 界面记录+不影响作业 |

## 9.2 智能安全供电管理

### 9.2.1 电池管理系统（BMS）

| 项目 | 参数 |
|---|---|
| 电池类型 | 磷酸铁锂（LiFePO4） |
| 电池组电压 | 48V DC |
| 电池容量 | >=40Ah |
| BMS芯片 | 智能BMS芯片（精度<=1%） |
| 均衡功能 | 被动均衡，防止电芯不一致 |
| 保护功能 | 过充、过放、过流、短路、过温 |
| 通信接口 | RS485 / CAN |

### 9.2.2 BMS关键安全参数

| 项目 | 保护阈值 |
|---|---|
| 过充保护电压 | 单芯3.65V |
| 过放保护电压 | 单芯2.5V |
| 过流保护电流 | >=30A |
| 短路保护 | 短路电流>=50A，<=1ms切断 |
| 充电过温 | >=55°C停止充电 |
| 放电过温 | >=60°C停止放电 |
| 低温保护 | <=-10°C禁止大电流放电 |

### 9.2.3 BMS管理板温度监控

BMS管理板内置温度传感器，实时监控板载电路温度：

| 温度范围 | 状态 |
|---|---|
| <50°C | 正常，绿色指示 |
| 50°C–60°C | 轻微发热，黄色预警 |
| 60°C–70°C | 过热，降载运行，橙色告警 |
| >70°C | 严重过热，停止充电/放电，红色告警 |

### 9.2.4 三级供电保护机制

| 保护级别 | 实现方式 | 说明 |
|---|---|---|
| 第一级：电池BMS | 电芯级硬件保护 | 独立于主控，硬件直接切断 |
| 第二级：供电管理板 | 系统级保护 | 监测总电流/电压，过流/过压切断 |
| 第三级：主控软件 | 逻辑层保护 | 软件判断异常后下发断电指令 |

### 9.2.5 全节点用电监控

| 节点 | 监控方式 | 说明 |
|---|---|---|
| 主控计算机 | 电流传感器+软件监控 | 电压/电流实时采样 |
| AI加速单元 | 板载传感器 | GPU温度/功耗软件读取 |
| 云台驱动 | 电机驱动器反馈 | 电流/温度状态字 |
| 行走驱动 | 电机驱动器反馈 | 电流/温度状态字 |
| 照明系统 | LED驱动电流反馈 | 亮度降低时自动报告 |
| 相机系统 | GigE供电监控 | PoE电流监控 |

## 9.3 续航保障设计

| 设计项 | 参数 |
|---|---|
| 续航时间 | >=4小时（标准作业工况） |
| 电池热插拔 | 支持（主控不重启） |
| 应急电容 | 断电时维持>=5s，用于安全制动和位置保存 |
| 续航估算 | 实时预估剩余续航时间，显示在手持终端 |
| 低电量保护 | 电量<10%时强制返航，>5%禁止执行新任务 |

## 9.4 防护设计

### 9.4.1 整体防护等级

| 部件 | 防护等级 |
|---|---|
| 机器人整体 | IP67（防尘防水防油污） |
| 主控计算机 | IP54 |
| 相机/镜头 | IP67 |
| 激光雷达 | IP67 |
| 云台电机 | IP67 |
| 线缆连接器 | IP67 |
| 手持终端 | IP65 |

### 9.4.2 环境适应性

| 环境因子 | 适应措施 |
|---|---|
| 粉尘/油污 | 全密封设计，镜头处增加防污罩 |
| 水/泥浆 | 整车IP67，线缆接头灌胶密封 |
| 高低温 | 工业级器件，-20°C~+60°C工作范围 |
| 振动/冲击 | 铝合金车体+减震垫，抗震设计 |
| 电磁干扰 | 屏蔽线缆，金属外壳，EMC设计 |
| 潮湿/霉菌 | 防潮涂层，防霉菌材料 |

## 9.5 故障自诊断与恢复

| 功能 | 说明 |
|---|---|
| 开机自检 | 启动时自动检测各模块，发现问题提示 |
| 运行时自检 | 周期性自检，发现异常报警 |
| 故障码 | 故障时显示故障码，查手册定位问题 |
| 自动恢复 | 软件异常自动重启，看门狗保护 |
| 断点保存 | 断电前自动保存任务状态，来电续接 |
