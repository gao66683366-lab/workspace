## 二、TB/T 3355-2023 详细说明

### 2.1 基础信息

| 项目 | 内容 |
|------|------|
| 标准号 | TB/T 3355-2023 |
| 中文名称 | 轨道检测 轨道几何状态动态检测 |
| 英文名称 | Track inspection — Track geometry dynamic inspection |
| 发布日期 | 2023-02-27 |
| 实施日期 | 2023-09-01 |
| 代替标准 | TB/T 3355-2014 |
| 中国标准分类号 | S11 |
| 国际标准分类号 | 45.080 |
| 归口单位 | 中国铁道科学研究院集团有限公司标准计量研究所 |
| 批准发布部门 | 国家铁路局 |
| 适用范围 | 1435mm标准轨距新建、改建铁路线路验收和运营铁路线路评价 |

---

### 2.2 核心术语定义

| 术语 | 定义 |
|------|------|
| 轨道几何不平顺 | 轨道几何形位参数与标准值的偏差 |
| 高低 | 钢轨顶面或线路中心线竖向的凹凸不平 |
| 轨向 | 轨道纵向的平面位置偏移 |
| 轨距 | 同一横截面左右钢轨间的最短距离 |
| 水平 | 左右钢轨顶面高差 |
| 三角坑 | 一定基长间隔的水平差变化率 |
| 复合不平顺 | 轨向与水平的叠加不平顺 |
| 轨距变化率 | 轨道延长方向基长间隔的轨距代数差与基长比值（‰） |
| 轨道质量指数(TQI) | 单元长度轨道的7项单项标准差之和 |

---

### 2.3 检测项目

**动态检测项目（10项）**：
1. 高低
2. 左轨向
3. 右轨向
4. 轨距
5. 水平
6. 三角坑
7. 复合不平顺
8. 轨距变化率
9. 车体垂向振动加速度
10. 车体横向振动加速度

---

### 2.4 评价方法

**局部峰值管理**：
- 采用扣分法
- 以整公里为单位计算扣分总数

**区段均值管理**：
- 采用TQI轨道质量指数
- 单元长度：200m

## 五、备注
