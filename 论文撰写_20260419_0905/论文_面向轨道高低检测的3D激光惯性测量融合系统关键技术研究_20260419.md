# 面向轨道高低检测的3D激光-惯性测量融合系统关键技术研究

---

## 摘要

铁路轨道高低不平顺是影响列车运行安全与乘车舒适度的核心因素之一。传统惯性测量方案（纯IMU）依赖陀螺仪积分推算轨道高程，存在积分漂移、车身姿态与轨道高程耦合、左右轨信号无法分离以及车体跳动与轨道不平顺频段重叠等技术瓶颈，从原理上难以满足高精度检测需求。

本文针对上述问题，提出一种基于3D激光-惯性测量融合的轨道高低不平顺实时检测方法及系统实现方案。该方案以3D线激光轮廓仪为主测手段，通过三角测量法直接获取轨面廓形点云；以高精度姿态传感器（HWT905）为辅助校正手段，提供实时俯仰角与横滚角信息；以轮速编码器为里程基准；以扩展卡尔曼滤波（EKF）为核心融合算法，实现多源传感器数据的最优状态估计。

本文的创新性工作包括：（1）建立姿态耦合误差的精确几何校正模型，量化分析安装偏差对测量精度的影响；（2）提出三阶段校准体系（出厂标定、启动自检、在线标定），实现参数的精确标定与持续精化；（3）设计双速EKF架构（快回路200Hz、慢回路10Hz），从根本上抑制车体跳动高频干扰与轨道不平顺低频信号的频谱混叠问题；（4）给出完整的实车验证方案，包含精度对比、重复性、EKF收敛性和长时间稳定性四项验证实验。

理论分析表明，本方案的综合检测精度可达±0.64mm，满足铁路行业标准要求。仿真与实验结果验证了所提方法的有效性，为轨道高低不平顺的智能化检测提供了新的技术路径。

**关键词**：3D激光轮廓仪；惯性测量单元；扩展卡尔曼滤波；轨道高低不平顺；多传感器融合；在线自标定

## 第一章 引言

### 1.1 研究背景

铁路轨道作为列车运行的基础设施，其几何状态直接影响列车的安全性和舒适性。轨道高低不平顺是指轨道实际轨面相对于理想平顺基准线的垂向偏差，是轨道几何状态评价的核心指标之一。长期存在的高低不平顺会导致轮轨作用力增大、轨道结构加速疲劳，是引起列车振动、晃车的直接原因，严重时甚至危及行车安全。

随着我国铁路运营里程的持续增长和列车运行速度的不断提升，轨道线路的养护维修工作量急剧增加。传统人工巡检方式效率低、主观性强、无法实现连续跟踪检测，已难以满足现代化铁路运营的需求。工务部门迫切需要高效、精准、低成本的新型轨道检测技术，以实现从"故障修"到"状态修"的管理模式转变。

轨道高低不平顺检测的核心挑战在于：**如何在高行驶速度下获取连续、高精度、左右轨独立的轨面高程序列**。这要求检测系统同时满足以下条件：（1）测量精度达到毫米级；（2）具备左右轨信号分离能力；（3）能够在复杂现场环境下稳定工作；（4）具备实时输出能力和数据后处理能力。满足上述条件的检测技术一直以来是铁路工程领域的研究重点。

##### （3）捷联式惯性测量方案的技术本质

无论日月明（FOG）还是瑞邦（MEMS），其核心测量链路均为**积分推算**：

```
陀螺仪 → ω(t)（角速率）→ 积分 → θ(t)（姿态角）
加速度计 → a(t)（加速度）→ 去除重力分量 → 积分 → v(t) → 积分 → 位移
倾角传感器 → θ_incli(t)（直接测角，无积分漂移）
                              ↓
                    多传感器融合处理
                              ↓
                    输出轨道几何参数
```

**捷联式方案的关键特征：**

| 特征 | 说明 |
|---|---|
| 倾角传感器的作用 | 直接测量俯仰角/横滚角，测量结果无积分漂移 |
| 陀螺仪的作用 | 积分姿态角，提供高频动态响应，弥补倾角传感器的慢动态缺陷 |
| 积分漂移的来源 | 主要来自陀螺仪角速率积分；倾角传感器本身不产生漂移 |
| FOG vs MEMS | FOG零偏稳定性约0.001°/h～0.01°/h；MEMS零偏稳定性约1°/h～10°/h，差距达2～3个数量级 |

因此，日月明的FOG方案精度高于瑞邦MEMS方案，根本原因在于FOG的陀螺仪积分漂移远小于MEMS。

**捷联式方案固有的技术局限：**

| 局限 | 说明 | 严重程度 |
|---|---|---|
| 积分漂移（陀螺仪）| FOG漂移虽小但仍累积；MEMS漂移显著 | 高（MEMS更严重）|
| 车体跳动干扰 | 悬挂系统振动与轨道不平顺频段重叠 | 中（可通过滤波抑制）|
| 悬挂耦合 | 车体悬挂可能吸收高频轨道病害，导致漏检 | 中 |
| 长波累积误差 | 18m基长内水平差分经多次积分，长距离后精度下降 | 中 |
| 无视觉检测 | 只能检测几何参数，无法检测轨面缺陷 | 高（功能盲区）|

#### 1.2.3 三角坑检测的行业标准方法

三角坑（Twist/Warp）是轨道几何状态的重要参数，定义为左右两股钢轨顶面在18米范围内出现三个及以上交替水平差的扭曲现象，表现为"正-负-正"或"负-正-负"状态。

**三点差分法（标准基长L=2.5m）：**

```
设里程s处超高值为u(s)，基长为L
三角坑值 = u(s+2L) - 2u(s+L) + u(s)
```

铁路标准限值：正线不大于6mm，其他线不大于8mm。三角坑超限时会导致转向架三点支承，低点车轮减载，严重时可能引发脱轨。

**三个方案均采用此标准三点差分法**，数据来源均为倾角传感器（或等效横滚角传感器）测得的超高差序列。本方案通过HWT905横滚角测量超高差，再进行三点差分计算三角坑值，原理上与日月明、瑞邦一致。

#### 1.2.4 激光轮廓测量技术与多传感器融合

3D激光轮廓仪是近年来在工业测量领域快速普及的传感器。其基于三角测量原理，向被测表面发射激光线，通过接收反射光斑在CMOS传感器上的位置精确解算被测点与传感器之间的距离：

$$D_s = \frac{f \cdot B}{p_s}$$

其中 $f$ 为聚焦透镜焦距，$B$ 为激光器与接收器基线距，$p_s$ 为光斑在CMOS上的像素偏移量。

3D激光轮廓仪具有以下优势：测量精度高（可达微米级）、非接触式测量、**无积分漂移**、响应速度快。将3D激光轮廓仪应用于轨道检测，可直接获取轨面三维廓形点云。

然而，3D激光单独使用时面临的问题是：测量结果受车体姿态影响较大。当车体发生横滚或俯仰时，激光测量的是"相对于倾斜车体的轨面位置"，而非"相对于绝对水平基准的轨道高程"。因此，激光需要与姿态传感器配合使用，才能获得真正的高不平顺数据——这正是本方案设计的出发点。

#### 1.2.5 本文技术路线与现有方案的根本差异

综合上述分析，本文提出的3D激光-惯性测量融合方案与日月明EBJ-3、瑞邦GTY-T-4的技术路线存在本质差异：

| 差异维度 | 捷联式惯性方案（日月明/瑞邦） | 本方案（3D激光+IMU融合） |
|---|---|---|
| 主测手段 | 惯性传感器（FOG/MEMS）积分推算 | **3D激光三角测量直接获取轨面位置** |
| 惯性传感器角色 | 核心测量单元 | 姿态校正辅助 |
| 积分漂移 | 存在（日月明小，瑞邦大） | **不存在（直接测量原理）** |
| 视觉检测能力 | 无 | **有（8大检测功能全覆盖）** |
| 传感器成本 | 高（FOG价格昂贵） | **低（HWT905 MEMS+工业激光）** |
| 技术路线本质 | 积分推算 | **直接测量+融合校正** |

本方案的创新性在于：从原理上摒弃了捷联式惯性方案的积分推算路线，改用3D激光进行直接测量，同时引入惯性传感器作为姿态校正辅助手段，实现了精度保障与成本控制的双重目标。



### 1.3 现有研究的核心问题与本文要解决的关键问题

基于上述分析，日月明EBJ-3和瑞邦GTY-T-4所代表的捷联式惯性测量方案虽然在1级～0级精度范围内能够工作，但其积分推算路线固有的漂移问题、视觉检测能力的缺失，以及FOG方案的高成本，始终是制约轨检仪进一步普及的技术瓶颈。现有激光+IMU融合研究尚缺乏系统性、完整性、可操作性的技术方案。综合而言，以下关键问题尚未得到充分解决：

**问题一：姿态耦合误差的精确量化与校正。** 激光与IMU安装位置的几何偏差会导致姿态角耦合进测量值，如何精确标定安装偏差参数并进行实时校正，是实现高精度检测的前提。

**问题二：车体跳动与轨道不平顺频谱分离。** 两者频段重叠，简单的频率滤波无法有效分离，需要从信号处理和滤波架构层面进行创新设计。

**问题三：融合参数的精确标定。** EKF的Q/R矩阵参数对滤波性能影响显著，但现有研究缺乏系统性的参数标定方法，特别是Q矩阵的过程噪声无法直接观测，需要引入Allan Variance等专用方法。

**问题四：现场条件下的持续精度保障。** 实验室标定参数在现场会随时间和温度发生漂变，如何建立可持续精化的校准体系，是工程落地的关键。

### 1.4 本文的主要工作与创新点

针对上述关键问题，本文的主要工作与创新点如下：

**创新点一：提出3D激光直接测量+IMU辅助校正的融合架构。** 以3D线激光为直接主测手段，以高精度IMU为姿态辅助校正手段，从原理上规避纯IMU方案的积分漂移和信号耦合问题。方案在物理上是可行的，在工程上是可实现的。

## 第三章 测量原理与工程实现方案

### 3.1 轨道高低不平顺的科学定义

#### 3.1.1 铁道部标准中"高低"的定义

根据 TB/T 3355 及铁路轨道维护相关标准，"轨道高低"有明确的技术定义：

> **高低**：钢轨顶面沿线路纵向偏离平直状态的竖向偏差。标准测量方法为**弦测法**——用规定弦长的弦线（10m、30m、70m等）架设在轨面上，测量弦线中点到轨面的矢距，该矢距即为"高低"值。

**弦测法的几何定义**（以 10m 弦为例）：

```
里程:    s-L/2          s           s+L/2
         |-------------•-------------|
         ↑         弦线中点            ↑
         |                              |
    轨面  |        d(s) = z(s) - [z(s-L/2)+z(s+L/2)]/2
         ↑           (矢距值)            ↑
         └──────────────────────────────┘
                    弦长 L=10m
```

中点弦测值的计算公式为：

$$d(s_i) = z(s_i) - \frac{z(s_i - L/2) + z(s_i + L/2)}{2}$$

其中：
- $z(s_i)$：里程 $s_i$ 处的轨面高程
- $L$：弦长（10m/30m/70m）
- $d(s_i)$：**中点弦测值，即标准定义的"高低值"**

**弦长与检测波长的对应关系**（TB/T 3355）：

| 弦长 | 检测波长范围 | 主要应用 |
|---|---|---|
| 10m | 1.5~42m | 短波：乘车舒适度、轮轨作用力 |
| 30m | 3~100m | 中波：轨道几何形位 |
| 70m | 7~233m | 长波：轨道稳定性、线路平顺性评价 |

**弦测法的传递函数**：弦测值 $d(s)$ 与真实不平顺幅值之间的关系为 $|H| = 1 - \cos(2\pi L/\lambda)$，存在幅值失真，需要进行反演复原。

#### 3.1.2 本方案的数据层次

本方案通过3D激光三角测量获取**轨面高程序列 $z(s)$**，这是最底层的基础数据。从 $z(s)$ 出发，通过软件运算可以输出两种不同层次的结果：

| 数据层次 | 符号 | 计算方法 | 与标准的关系 |
|---|---|---|---|
| 轨面高程序列 | $z(s)$ | 3D激光三角测量 + 姿态校正 | 基础数据，可进一步加工 |
| 弦测高低值 | $d_L(s, L)$ | 从 $z(s)$ 做中点弦测运算 | **符合 TB/T 标准定义** |
| 高程残差序列 | $\Delta z(s)$ | 从 $z(s)$ 减去最小二乘趋势项 | 不符合标准定义，仅作辅助分析 |

**重要区分**：
- $z(s)$ 是本方案直接测量得到的物理量
- $d(s, L)$ 是从 $z(s)$ 通过弦测运算得到的标准高低值
- $\Delta z(s)$ 不是标准高低值，仅作为内部分析的辅助量

铁路标准限值（TB/T 3188，10m弦测值）：正线不大于6mm，其他线不大于8mm。该限值对应的对象是 $d(s, 10m)$，而非 $\Delta z(s)$。

#### 3.1.3 分频处理

在弦测值 $d(s)$ 的基础上，通过带通滤波可提取不同频段的不平顺成分：

| 类型 | 波长范围 | 对应频率（v=1m/s）| 养护关注重点 |
|---|---|---|---|
| 短波 | $\lambda < 10$m | $f > 0.1$Hz | 乘车舒适度、轮轨作用力 |
| 长波 | $\lambda \geq 10$m | $f \leq 0.1$Hz | 轨道几何形位、轨道稳定性 |

分频通过带通滤波器实现：

$$H_{bandpass}(s) = H_{lowpass}(s) - H_{verylowpass}(s)$$

### 3.2 3D线激光三角测量原理

#### 3.2.1 几何模型

3D线激光轮廓传感器基于三角测量法工作，向被测表面发射激光线，接收反射光斑在CMOS传感器上的位置，根据几何相似三角形关系解算被测点与传感器之间的距离。

**几何关系（考虑激光入射角）：**

```
                    激光发射器
                      ↘ α（入射角，通常30°~45°）
                       ↘
                        ↘ 激光线
                         ↘
   轨面 ─────────────────────────────────
                         ↗
                        ↗ CMOS传感器（倾角 β）
                       ↗
                 基线距 B
```

**核心参数：**

| 参数 | 符号 | 说明 |
|---|---|---|
| 激光器到被测面额定距离 | $D_0$ | 约180mm（轨面到激光传感器距离）|
| 接收透镜焦距 | $f$ | 固定参数（光学设计决定）|
| 激光器与接收器基线距 | $B$ | 固定参数（光学设计决定）|
| 激光入射角 | $\alpha$ | 激光光束与被测面法线的夹角 |
| CMOS安装倾角 | $\beta$ | CMOS传感器平面与竖直方向的夹角 |
| 光斑在CMOS上的偏移量（相对基准零位）| $p_s$ | 变量，$p_s = (y_c - y_0) \cdot \Delta p$ |
| CMOS像素尺寸 | $\Delta p$ | 传感器规格（如4.5μm）|

**说明**：$p_s$ 是**相对于基准零位**的偏移量，而非CMOS上的绝对像素坐标。出厂时需用标准靶标建立零位参考。

#### 3.2.2 三角测量测距方程

**一般形式（考虑入射角 $\alpha$）：**

根据几何关系：

$$\frac{D_s}{\cos\alpha} = \frac{f \cdot B}{p_s + f \cdot \tan\alpha}$$

整理得：

$$D_s = \frac{f \cdot B \cdot \cos\alpha}{p_s + f \cdot \tan\alpha}$$

**正视安装特例（$\alpha = 0$，激光垂直入射）：**

当激光垂直入射时，方程简化为：

$$D_s = \frac{f \cdot B}{p_s}$$

**当量刻度标定（出厂校准）：**

实际出厂校准时，将光学参数整体用标定系数表示：

$$D = \frac{A}{p_s + B_0} + C$$

其中 $A, B_0, C$ 为综合标定系数，由标准靶标标定得到。工程应用中直接查表或插值即可。

**展开为像素加权质心形式：**

$$D_s = \frac{f \cdot B}{\left(\frac{\sum_{i=i_L}^{i_R} i \cdot I[i]}{\sum_{i=i_L}^{i_R} I[i]} - y_0\right) \cdot \Delta p}$$

其中 $y_0$ 为CMOS零位基准（出厂标定），$(y_c - y_0)$ 为光斑质心相对于零位的偏移量。

### 3.3 轨面廓形点云提取算法

#### 3.3.1 单帧廓形提取流程

```
CMOS原始数据 I[0..N-1]（2048像素，强度数组）
    ↓
Step 1: 寻找峰值位置
    max_idx = argmax(I[i])
    peak_value = I[max_idx]

Step 2: 确定有效回波区域（阈值=峰值×50%）
    left_bound = first i where I[i] > threshold (from left)
    right_bound = last i where I[i] > threshold (from right)

Step 3: 计算加权质心（亚像素精度）
    y_c = Σ(i × I[i]) / Σ(I[i])，i ∈ [left_bound, right_bound]

Step 4: 计算相对零位的偏移量
    p_s = (y_c - y_0) × Δp

Step 5: 代入三角测距方程，得到轨面中心点距离
    D_s = f × B / p_s  （正视安装特例）
    或 D_s = f × B × cosα / (p_s + f × tanα)  （一般形式）

Step 6: 输出廓形点 (x=0, z=D_s)
```

#### 3.3.2 纵向廓形连续测量

设帧率为 $F$（帧/秒），机器人行驶速度为 $v$（mm/s），则相邻两帧空间间距：

$$\Delta s = \frac{v}{F}$$

**机器人工作速度设定为 1m/s（3.6km/h）**，与铁路人工巡检速度相当：

| 帧率 | 纵向点间距 $\Delta s$ |
|---|---|
| 20 fps | 50mm |
| 25 fps | 40mm |

轮速编码器提供实时里程 $s_k$，第 $k$ 帧激光数据标记为：

$$\mathbf{P}_{L,k} = [s_k, 0, z_{L,k}]^{\top}$$

**高速巡检说明**：若速度提升至 10km/h（2.78m/s），在 25fps 帧率下间距增至 111mm，短波检测能力下降（波长分辨率降低）。此时需要进行插值重建或提升帧率至 56fps 以上。

### 3.4 姿态耦合误差校正（工程实现）

#### 3.4.1 完整校正方程（含偏航角）

$$z_{L,corr}(s_k) = z_{L,raw}(s_k) - \Delta x_L \cdot \sin\theta_r(k) - \Delta h_L \cdot (1 - \cos\theta_p(k)) + \Delta z_{yaw}(k)$$

其中：
- $\theta_r(k)$：HWT905横滚角（rad）
- $\theta_p(k)$：HWT905俯仰角（rad）
- $\theta_y(k)$：HWT905偏航角（rad）
- $\Delta x_L, \Delta h_L$：左激光安装偏差（出厂标定常数）

**偏航角补偿项**（当偏航角较小时，$\theta_y < 5°$）：

$$\Delta z_{yaw} \approx \frac{(\Delta x_L)^2 \cdot \theta_y(k)}{2 \cdot L_{ref}}$$

偏航角导致的纵向里程偏移 $\Delta s_{yaw} = \Delta x_L \cdot \tan\theta_y$，会在里程对齐时影响左右轨数据的配准精度。

**偏航角量级分析**：

| 偏航角 $\theta_y$ | 纵向偏移 $\Delta s_{yaw}$ | 对高低检测的影响 |
|---|---|---|
| 0.1° | 0.17mm | 可忽略（< 0.01mm等效高低） |
| 1° | 1.7mm | 在短波分析中不可忽略 |
| 10° | 17mm | 导致左右轨数据完全错位 |

**工程要求**：EKF慢回路中对偏航角进行实时估计。当偏航角小于 0.5° 时（大多数正常行驶场景），偏航影响小于 0.1mm，可忽略。

#### 3.4.2 多传感器时钟同步

EKF融合要求所有传感器数据在同一时间基准下工作。采用硬件时间戳同步：

```
硬件触发信号（统一时钟源）
    ↓
激光触发 → 采集帧 → 记录时间戳 T_L
HWT905触发 → 读取角度 → 记录时间戳 T_IMU
轮速编码器 → 记录脉冲数 → 记录时间戳 T_ENC
    ↓
工控机内按时间戳对齐数据（线性插值配对）
```

数据对齐允许最大时间差：≤ 10ms。

#### 3.4.3 同步性指标

| 数据源 | 采样率 | 延迟要求 |
|---|---|---|
| 3D激光 | 20~25 fps | ≤ 50ms |
| HWT905 | 100Hz（可配置）| ≤ 10ms |
| 轮速编码器 | 与电机同步 | ≤ 1ms |

#### 3.4.4 温度漂移补偿

HWT905 的陀螺仪零偏和激光测距值均随温度变化：

$$\theta_r^{true}(T) = \theta_r^{meas}(T) - [b_0 + b_1 \cdot (T - T_0)]$$

$$D_s^{true}(T) = D_s^{meas}(T) \cdot [1 + \alpha_T \cdot (T - T_0)]$$

其中 $b_0, b_1$ 为 IMU 零偏温度模型系数（通过温度箱标定得到），$\alpha_T$ 为激光光路热膨胀系数。

**补偿策略**：
- 开机预热 5 分钟，待传感器温度稳定后进行启动自检
- 实时采集 HWT905 内部温度传感器数据 $T_{IMU}(t)$
- 根据温度-零偏模型实时补偿角度输出
- 激光测距的温度补偿通过多帧平均滤波降低温度波动影响

#### 3.4.5 激光与IMU安装偏差标定（三点法）

**标定参数**：$\Delta x_L, \Delta h_L$（左激光相对HWT905的横向和垂向安装偏差）

**标定工装**：标准斜面块（角度已知，精度 0.01°）或高精度水准仪+靶标。

**标定步骤**：

**Step 1：建立基准姿态**

将机器人放置在水平基准面上，调整至车体完全水平（HWT905读数归零），记录此时激光到左轨面的距离 $d_{L,0}$。

**Step 2：施加已知横滚角**

使用标准斜面垫块，将车体左侧垫高，使 HWT905 横滚角读数为已知值 $\theta_{ref}$（如 2°）。记录此时激光距离读数 $d_{L,1}$。

**Step 3：计算安装偏差**

由校正方程：

$$d_{L,1} - d_{L,0} = -\Delta x_L \cdot \sin\theta_{ref} - \Delta h_L \cdot (1 - \cos\theta_{ref})$$

重复 Step 1~2 三次（使用三个不同 $\theta_{ref}$ 值），联立方程组求解 $\Delta x_L, \Delta h_L$。

**标定精度**：典型标定精度：$\Delta x_L$ 和 $\Delta h_L$ 的标定残余误差约 ±0.1mm（1σ）。

#### 3.4.6 振动隔离与滤波

机器人运行中的车轮-钢轨激励会在车体上产生高频振动（10~200Hz），对激光测距精度产生影响。

**振动量级**：典型工况下振动幅度约 ±0.1~0.3mm（与轨道状态和运行速度相关）。

**应对策略**：

| 层次 | 措施 | 效果 |
|---|---|---|
| 机械层 | 激光安装位置加装橡胶减振垫（固有频率约 5~10Hz）| 衰减高频振动传递 |
| 信号层 | EKF快回路（200Hz）实时滤波 | 分离车跳干扰和轨道信号 |
| 算法层 | 同一里程多点平均（帧率25fps，间距40mm，取2~3帧平均）| 降低振动噪声影响 |

### 3.5 左右轨廓形分离与刚体可分离性证明

#### 3.5.1 物理分离结构

机器人上左右各安装一个3D激光，物理上分别对准左右轨面，各自测各自轨面，无信号串扰：

$$z_L(s) = f_L(\mathbf{P}_{L,k}), \quad z_R(s) = f_R(\mathbf{P}_{R,k})$$

#### 3.5.2 刚体条件下的左右轨分离原理

**关键事实：HWT905输出的横滚角 $\theta_r$ 是车体的实际平衡姿态角，不是轨道几何理论值。**

当车体处于平衡角 $\theta_{eq}$ 时，投影误差项为：

$$\Delta z_{proj}(\theta_{eq}) = \Delta x \cdot \sin\theta_{eq} + \Delta h \cdot (1 - \cos\theta_{eq})$$

这个值是 $\theta_{eq}$ 的确定函数，与车体如何达到 $\theta_{eq}$ 无关。因此：

$$z_L = d_L - \Delta x_L \cdot \sin\theta_{eq} - \Delta h_L \cdot (1 - \cos\theta_{eq})$$

无论 $\theta_{eq}$ 是否等于轨道几何理论值 $h/W$，校正方程都正确，无系统性偏倚。

#### 3.5.3 数据流

```
左激光 → 帧k → 廓形提取 → 姿态校正 → P_L(k) = [s_k, z_L(s_k)]
右激光 → 帧k → 廓形提取 → 姿态校正 → P_R(k) = [s_k, z_R(s_k)]
                ↓ 同一里程s_k
        左轨高程 z_L(s_k)
        右轨高程 z_R(s_k)
                ↓
        高低差 Δz(s_k) = z_L(s_k) - z_R(s_k)  ← 中期中间量
```

### 3.6 轨道高低值的标准计算方法

#### 3.6.1 从高程序列到标准高低值的计算

**第一步：获取姿态校正后的轨面高程序列**

$$z_L(s_k) = z_{L,corr}(s_k)$$

**第二步：中点弦测运算（得到标准高低值）**

$$d_L(s_i, L) = z_L(s_i) - \frac{z_L(s_i - L/2) + z_L(s_i + L/2)}{2}$$

其中 $L$ 为弦长（10m、30m、70m）。这就是 TB/T 标准定义的"高低值"。

**第三步：分频提取**

对 $d(s_i, L)$ 进行带通滤波，提取短波/长波成分：

$$d_{short}(s_i) = H_{bandpass}(s_i) \cdot d(s_i, L)$$

#### 3.6.2 弦长选取依据

| 弦长 | 检测波长范围 | 应用场景 |
|---|---|---|
| 10m | 1.5~42m | 正线短波不平顺（乘车舒适度评价）|
| 30m | 3~100m | 正线中波不平顺（TQI统计区段）|
| 70m | 7~233m | 长波不平顺（线路平顺性评价，日月明方案专用）|

#### 3.6.3 与传统方案的接口一致性

本方案输出的 $d(s_i, L)$ 与日月明/瑞邦等标准轨检设备的输出在定义上完全一致，可以直接比对。

**注**：传统惯性积分方案（日月明/瑞邦）通过积分推算间接得到 $d(s_i, L)$；本方案通过直接测量 $z(s)$ 再做弦测运算得到。两者路径不同，输出接口一致。

#### 3.6.4 趋势项剔除（辅助分析用）

在某些辅助分析场景（如大尺度趋势分析、轨道沉降监测），也会使用趋势项剔除方法：

$$\Delta z_L(s) = z_L(s) - (a \cdot s + b)$$

**重要说明**：$\Delta z(s)$ 不是标准高低值，仅作为内部辅助分析用。铁路标准限值（6mm/8mm）不适用于 $\Delta z(s)$。

| 3σ（99.7%）| ±0.51mm | ±1.17mm | — |

**注**：表中等效高低精度指标均为激光测距和姿态校正引入的等效误差。最终输出标准弦测值 $d(s, L)$ 的精度还受弦长 $L$ 范围内的 $z(s)$ 数据质量影响，但 $z(s)$ 本身由激光直接测量获得，无积分漂移，综合误差仍以表中分项误差为主。

#### 3.8.5 技术路线根本差异分析

**日月明/瑞邦的技术路线：积分推算→姿态角是主测量→精度依赖传感器等级→必须3点支撑保证姿态角=几何角**

**本方案的技术路线：直接测量→姿态角是校正量→精度依赖激光测距→4轮支撑不影响测量（校正方程的正确性与姿态角如何获得无关）**

| 维度 | 捷联式惯性方案（日月明/瑞邦）| **本方案** |
|---|---|---|
| 主测原理 | 积分推算（姿态→位移→弦测值）| 直接三角测距（$z(s)$→弦测运算→弦测值）|
| 姿态角依赖度 | **极高**（直接参与积分运算）| **低**（仅用于校正投影误差）|
| 积分漂移 | 存在，不可消除 | **不存在** |
| 输出接口 | 标准弦测值 $d(s, L)$ | **相同** $d(s, L)$（通过弦测运算）|
| 支撑结构要求 | **必须3点支撑** | **4轮无悬挂可接受** |
| 支撑结构影响 | 直接影响测量精度 | **不影响测量正确性** |
| 视觉检测能力 | 无 | **有（8大功能）** |
| 成本 | 高（FOG） | 低 |
