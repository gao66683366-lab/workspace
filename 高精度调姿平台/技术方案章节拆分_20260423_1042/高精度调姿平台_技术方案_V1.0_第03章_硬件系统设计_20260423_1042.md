# 3. 硬件系统设计

## 3.1 硬件系统总体架构

```
高精度调姿平台硬件系统
│
├── 1. Stewart并联平台（执行层）
│   ├── 上平台（静平台）
│   ├── 下平台（动平台）
│   └── 6组伺服电动缸（含伺服电机）
│
├── 2. 感知系统
│   ├── 单点测距传感器×2（底部朝下，测轨道高程）
│   └── HWT905姿态传感器（平台中心）
│
├── 3. 计算与控制系统
│   ├── Jetson AGX Orin 主控模块
│   └── EtherCAT总线控制器
│
└── 4. 伺服驱动系统
    └── 6轴伺服驱动器
```

## 3.2 Stewart并联平台结构

### 3.2.1 平台结构参数

| 参数 | 数值 | 说明 |
|---|---|---|
| 下平台直径 | φ200 mm | 铝合金，轻量化设计 |
| 上平台直径 | φ300 mm | 钢结构，承载基座 |
| 支链数量 | 6组 | 均匀分布，120°间隔 |
| 电动缸行程 | 20 mm | 紧凑型伺服电动缸 |
| 电动缸额定推力 | ≥ 200 N | 满足2kg负载+动态惯性力 |
| 万向节类型 | 虎克铰/球铰混合 | 两端多自由度连接 |
| 上平台重量 | ≤ 3 kg | 静平台，铝合金 | 含传感器安装支架 |

### 3.2.2 运动学参数

| 参数 | 数值 |
|---|---|
| 最大横滚角 | ±15° |
| 最大俯仰角 | ±15° |
| 最大偏航角 | ±10° |
| 最大垂直位移 | ±10 mm |
| 重复定位精度 | ±0.02 mm |

### 3.2.3 电动缸选型

| 项目 | 参数 |
|---|---|
| 类型 | 伺服电动缸（无杆式） |
| 行程 | 20 mm |
| 额定推力 | 200 N |
| 最大速度 | 50 mm/s |
| 响应时间 | ≤ 1 ms |
| 传动精度 | ±0.01 mm |
| 防护等级 | IP65 |

## 3.4 计算与控制系统

### 3.4.1 主控模块

| 项目 | 参数 |
|---|---|
| 型号 | Jetson AGX Orin |
| 算力 | 32 TOPS（INT8） |
| 内存 | 32 GB LPDDR5 |
| 用途 | EKF融合算法、逆运动学求解、控制策略 |
| 实时性 | Linux PREEMPT_RT补丁，≤1ms调度周期 |

### 3.4.2 实时控制器

| 项目 | 参数 |
|---|---|
| 类型 | EtherCAT主站控制器 |
| 控制周期 | 1 ms（1000Hz） |
| 通信 | EtherCAT硬实时总线 |
| 用途 | 6轴伺服驱动指令下发 |

### 3.4.3 控制链路延迟分解

| 环节 | 延迟 | 说明 |
|---|---|---|
| 测距采样 | 0.5 ms | 2000Hz，每周期0.5ms |
| 数据预处理 | 0.5 ms | 滤波、格式转换 |
| EKF融合 | 1.0 ms | 状态估计 |
| 逆运动学求解 | 0.5 ms | 6缸目标长度计算 |
| 指令下发 | 0.5 ms | EtherCAT通信 |
| 伺服响应 | 1.0 ms | 驱动器接收+响应 |
| 电动缸机械响应 | 1.0 ms | 机械惯性响应 |
| **全链路总计** | **≤ 5 ms** | 端到端延迟 |
