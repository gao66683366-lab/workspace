# 面向铁路线路的智能视觉与多模态感知融合综合检测系统研究与实现

## 第3章 硬件系统设计

### 3.1 传感器配置选型依据

硬件设计的核心是传感器的选型与配置。本节从每项检测功能的物理测量原理出发，论证传感器配置的合理性。

**轨面缺陷检测（功能01）** 需要获取轨面高清图像，用于识别裂纹、掉块、凹陷等缺陷。2D工业相机配合高亮度LED补光，可在0.5m/s速度下保证图像无运动模糊（曝光时间<1/500s），2448×2048分辨率可检测亚毫米级缺陷。GigE Vision协议支持长距离稳定传输，适合铁路现场部署。

**道钉/螺栓状态检测（功能02）** 同样依赖图像采集，但检测对象为轨枕表面的紧固件，侧重检测缺失和松动两种状态。归一化相关系数（ZNCC）模板匹配方法对图像质量要求与功能01相当，共享同一相机模组。

**3D钢轨轮廓检测（功能03）** 和**钢轨波磨检测（功能04）** 需要获取轨面三维点云数据。3D线激光传感器通过三角测量原理重建轨头廓形，扫描频率20000Hz、每轮廓3200点，可精确捕捉磨耗和波磨的空间形态。垂直朝下安装可避免斜射光斑变形。

**轨距检测（功能06）** 要求直接测量左右轨内侧距离，测距传感器矩阵通过红外相位式测距直接获取轨距值，与车体姿态无关，避免了横滚角对测量结果的干扰。

**水平检测（功能07）** 通过姿态传感器HWT905的横滚角数据计算超高差，无需额外传感器，200Hz采样率可满足实时性要求。

**高低检测（功能08）** 需从包含车体振动的原始信号中提取慢变的轨道高程信号。单点测距传感器以2000Hz采样率捕捉距离变化，姿态传感器以200Hz快回路校正俯仰角影响，两者融合后输出稳定的高程估计值。

### 3.2 感知系统配置

传感器配置如下：2D工业相机6台（2448×2048px，20fps，GigE Vision），分三组分别拍摄轨面（2台）、道钉/螺栓（2台）、焊缝（2台）；3D线激光传感器2台（20000Hz，IP67防护，垂直朝下）；HWT905姿态传感器1台（200Hz，三轴姿态角输出）；单点测距传感器2台（2000Hz）；测距传感器矩阵1套（红外相位式，直接测轨距）。

### 3.3 计算与通信系统

边缘计算单元采用Intel Core i7工业级工控机，IP54防护等级，部署在车体内部，负责视觉AI推理和点云处理算法实时运行。

三网物理隔离架构：EtherCAT总线连接伺服驱动器，千兆以太网专网连接相机和激光传感器，4G/5G无线网络负责数据上传和远程监控。三网独立运行，互不干扰。

### 3.4 传感器标定方案

传感器标定是确保测量精度的基础环节。本系统采用"工厂预标定+现场自标定"两级方案。工厂预标定在受控环境下建立传感器内参基准库（相机内参、激光三角测量参数、IMU零偏等）；现场自标定利用已知几何特征（标准轨距1435mm、轨面宽度80mm）作为约束，在每次出车前对关键参数进行在线校正，抑制温度漂移和机械松动引入的精度退化。

### 3.5 供电与运动平台

采用48V大容量蓄电池组供电，智能电源管理（BMS板温度监测、管理板温度监测、三级保护机制）保障系统长时间运行安全。BMS板温度和管理板温度双重监测确保供电系统始终运行在安全温度区间，三级保护（过流保护、过压保护、短路保护）防止意外故障扩散。

运动平台采用4轮驱动设计，通过伺服电机直接驱动，闭环控制保证运行速度稳定。平台可在标准轨距（1435mm）铁轨上稳定运行，轨道轮间距与轨距精确匹配，过曲线时自动调整前后轮转角以适应曲率半径变化。速度控制精度优于±0.05m/s，满足0.5m/s精细检测和1m/s常规检测两种模式的稳定切换。
