# 1. 项目概述与核心定位

### 1.1 核心目标

设计并实现一套基于实时测距反馈的Stewart并联机构高精度调姿平台，通过在平台底部安装单点测距传感器实时采集轨道高程信息，结合多源数据融合与逆运动学控制，实现平台在三维空间内的实时水平保持功能。

### 1.2 核心技术指标

| 指标名称 | 目标值 | 说明 |
|---|---|---|
| 平台负载 | ≤ 2 kg | 下平台（动平台）负载，含传感器 |
| 调姿精度（俯仰/横滚） | ≤ ±0.05° | 极高精度，两侧高差容限仅0.52mm（间距300mm） |
| 电动缸行程 | 20 mm | 紧凑设计，最大补偿角度约±3.8° |
| 响应延迟（全链路） | ≤ 5 ms | 核心瓶颈，从测距到执行端到端延迟 |
| 调姿响应带宽 | ≥ 2 Hz | 可补偿的轨道起伏频率上限（车速1m/s） |
| 传感器测距精度 | ±0.15 mm | 现场综合精度 |
| 传感器采样频率 | 2000 Hz | 保证高频检测能力 |
