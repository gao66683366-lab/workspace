## 05 人机交互与操控系统

### 一、设计理念：简单易用，降低操控成本

传统检测设备需要多人配合，专业培训、复杂操作。本系统以"简单易用"为核心理念，大幅降低操控门槛和使用成本：

- **3 步启机**：开机 → 设参 → 出发，非专业人员也可快速上手
- **单人操作**：一台设备一人管理，无需多人配合
- **无需专业培训**：图形化界面，故障提示清晰
- **全程自动**：启动后自动检测、自动报警、自动存储，无需人工值守

### 二、多模式操控系统

| 控制方式 | 功能说明 | 适用场景 |
|---|---|---|
| 物理金属按钮 | 实体按键，盲操作可靠 | 现场紧急操作、屏幕故障兜底 |
| 触摸屏控制 | 参数配置、状态监控、数据预览 | 作业前设置、作业中监控 |
| 无线遥控控制 | 远程启停、调速、返航 | 复杂现场、远距离安全操作 |
| 语音控制（规划） | 语音指令操控，解放双手 | 降低操作门槛，提升便捷性 |

### 三、一键定速巡航

一键定速巡航是本系统的核心自动驾驶辅助功能，确保检测过程速度恒定、数据质量一致。

| 功能 | 说明 |
|---|---|
| 一键启动 | 在触摸屏上选择"定速巡航"模式，输入目标速度（默认 1 m/s），一键启动 |
| 速度恒定 | 伺服驱动系统实时闭环控制，保持设定速度不变，不受路面坡度影响 |
| 数据质量保障 | 恒定速度确保 3D 激光和相机按固定间距采集数据，便于历史数据横向对比 |
| 异常降速保护 | 检测到前方障碍物或轨道异常时，自动降速甚至触发急停 |
| 远程可调 | 监控中心可远程调整巡航速度，适应不同区段检测要求 |

> 一键定速巡航让人工操作从"持续盯控"中解放出来，检测过程完全自动化，速度一致性由系统保证而非人工控制。

### 四、一键定点返航

一键定点返航是保障设备安全回收的核心功能，尤其在电量不足、任务异常或复杂现场环境下，确保设备安全返回。

| 功能 | 说明 |
|---|---|
| 一键触发 | 任何时刻按下"返航"按钮，或触摸屏上点击"定点返航"，立即启动返航程序 |
| 原路返回 | 机器人沿原采集轨迹反向行驶，精确返回出发点 |
| 自主避障 | 返航过程中自动检测前方障碍物，遇障自动停障 |
| 电量优先策略 | 当电量低于安全阈值时，系统自动强制触发返航，优先保证设备回收 |
| 数据保全 | 返航过程不丢失任何已采集数据，数据完整上传云端后任务闭合 |
| 远程返航 | 监控中心可远程下发返航指令，在现场人员无法操控时远程接管 |
| 里程到达自动返航 | 出发前设定本次检测里程（如 5km），到达设定里程后系统自动触发定点返航，无需人工操作，数据完整保存后任务闭合 |

> 一键定点返航从根本上解决了"设备放出去回不来"的安全隐患，是无人干预运行的重要保障。

### 五、为无人值守自主检测奠定技术基础

**一键定速巡航**和**一键定点返航**是本系统实现无人值守自主检测的两大核心功能支柱：

- **一键定速巡航**：确保机器人在设定速度下匀速行驶，检测数据采集间距一致、质量稳定，为全自动化检测流程提供速度基准。
- **一键定点返航**：配合里程设定，机器人完成检测任务后自动返回原点，无需人工介入，彻底打通"出发→检测→返回"的全闭环流程。

这两项功能共同构成了**自主导航、自主检测、自主归航**的完整自动化链条，为未来在铁路现场实现真正意义上的**无人值守全自动检测**提供了坚实的技术支撑。随着算法精度和设备可靠性的持续提升，系统可逐步实现夜间、恶劣天气条件下的自主作业，真正做到全天候无人干预运行。

### 七、操控成本降低体现

- **人力成本**：单人操作，减少 2~3 人/组的配置需求
- **培训成本**：图形化界面，无需专业人员操作
- **时间成本**：3 步启机，大幅缩短作业准备时间
- **维护成本**：远程参数调优 + 模型远程推送，减少到场维护次数

### 八、触摸屏操控详细设计

触摸屏是人机交互的核心界面，位于设备操控面板正面，支持实时状态监控与参数配置：

**界面布局：**

| 区域 | 内容 | 说明 |
|---|---|---|
| 顶部状态栏 | 速度/电量/信号/里程/时间 | 实时刷新，一目了然 |
| 主显示区 | 实时画面/缺陷标注/轨距值/姿态角 | 作业中主监控区 |
| 快捷操作区 | 一键巡航/一键返航/急停 | 盲操作大按钮设计 |
| 底栏功能键 | 任务/数据/设置/报警记录 | 分页导航 |

**交互流程：**

```
开机自检（3s） → 登录验证（可选PIN码） → 任务配置页 → 选择区段 → 一键出发
                                                                      ↓
                                                          实时监控（作业中）
                                                                      ↓
                                                          任务结束 → 数据上传 → 关机
```

**规格参数：**

| 项目 | 规格 |
|---|---|
| 屏幕尺寸 | 16 英寸工业电容触摸屏 |
| 分辨率 | 1920 × 1200 |
| 亮度 | 1000 cd/m²（阳光下可视） |
| 触控点数 | 10 点触控 |
| 防护等级 | IP65（正面） |
| 工作温度 | -20°C ~ 60°C |
| 安装方式 | 嵌入式面板安装 |
| 操作系统 | Linux 实时操作系统 |

| 按钮名称 | 颜色 | 功能 | 位置 |
|---|---|---|---|
| 急停按钮 | 红色（大） | 触发立即断电停车 | 设备顶部，醒目位置 |

| 停止按钮 | 黄色 | 正常停止作业 | 面板左侧 |
| 返航按钮 | 蓝色 | 触发定点返航 | 面板中部 |

**按钮规格：**

| 项目 | 规格 |

### 十、无线遥控器详细规格

无线遥控用于远距离操控或危险现场，减少操作人员暴露在轨道环境中的时间：

| 项目 | 规格 |
|---|---|
| 遥控距离 | 空旷场地 ≥ 500m |
| 控制通道 | 6 通道（启停/速度±/返航/急停/静音） |
| 通信频率 | 2.4GHz 跳频扩频（FHSS） |
| 加密方式 | 滚动码加密，防复制 |
| 显示屏 | 1.3 英寸 OLED，显示速度/电量/信号强度 |
| 电池续航 | 连续使用 ≥ 8 小时 |
| 充电接口 | USB-C |
| 防护等级 | IP66 |
| 重量 | 约 350g（含电池） |
| 急停反应时间 | < 50ms |

**遥控器安全机制：**
- 信号中断超过 1s 自动触发减速停车
- 信号丢失超过 3s 自动触发定点返航
- 多个遥控器同时操作时，以最后接收指令为准

### 十一、异常场景操控流程

以下异常场景均有明确操控逻辑，确保设备安全：

**场景 1：触摸屏死机**
```
检测：系统监测到触摸屏无响应超过 10s
处理：自动切换至物理按钮模式 → 声光提示"切换至按键模式" → 操作员使用物理按钮继续任务
```

**场景 2：遥控信号丢失**
```
检测：连续 1s 未收到遥控信号
处理：自动降速至 0.3 m/s → 等待 3s → 如未恢复，触发定点返航
```

**场景 3：电量不足（< 20%）**
```
处理：声光报警 → 屏幕提示"电量不足，建议返航" → 操作员确认
            → 如未操作，电量 < 10% 时自动强制触发定点返航
```

**场景 4：前方障碍物检测**
```
检测：激光测距传感器检测到前方 1.5m 内有障碍物
处理：自动降速至 0.3 m/s → 提示"前方障碍" → 操作员选择绕障或急停
```

**场景 5：GPS/北斗定位信号丢失（户外模式）**
```
处理：切换至里程计导航（不依赖GPS）→ 提示"定位信号丢失，使用里程计导航" → 继续任务，返航时使用记录轨迹返航
```

### 十二、操控系统可靠性保障

| 保障措施 | 说明 |
|---|---|
| 双模式冗余 | 触摸屏 + 物理按钮双套操控，任一故障不影响整体控制 |
| 看门狗监控 | 控制系统内置看门狗，异常自动复位 |
| 紧急停车回路 | 急停按钮独立于主控制系统，硬线连接，最优先响应 |
| 操作日志记录 | 所有按键/触控操作均记录时间戳，可追溯 |
| 权限分级 | 操作员 / 管理员 / 调试模式三级权限，防止误操作 |
