## 已确认的重要项目口径（高优先级保留）
- 机器人运行在铁轨上，4 个轮子。
- 速度：3 km/h = 0.833 m/s = 833.3 mm/s。
- 每天运行 2 小时，约 6 km。
- 道钉/螺栓节距：0.7 m = 700 mm。
- 所有 2D 工业相机、3D 线激光均垂直朝下。
- 两个 3D 线激光检测对象是左右钢轨的轨头轮廓。
- 姿态参数传感器用于检测：水平、高低、轨向。
- 6 个 2D 相机安装在铁路小车车架上。
- 拍轨面的 2D 相机离钢轨约 180 mm。
- 拍道钉螺栓的 2D 相机离轨面约 270 mm。
- 钢轨轨面实际宽度约 70.8 mm，工程计算统一按 72 mm 假定。
