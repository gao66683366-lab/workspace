# PROJECT_STRUCTURE

## 主项目：铁路线路智能检测机器人

项目根目录：`01-project/铁路线路智能检测机器人/`

### 项目结构

```
铁路线路智能检测机器人/
├── 原始技术文档/        # 原始设计文档（多版本）
├── 设计文档/           # 定稿版项目文档
├── 设备手册/           # 欧普特相机、基恩士3D线激光手册
├── 论文编撰/           # 发表材料、图表素材、文献综述、论文草稿
├── 会议记录/           # 会议记录文件
├── 数据分析/           # 分析报告、数据整理
├── 代码/               # 上位机软件、云端服务器、工具脚本
├── 图片与媒体/         # 图片、视频等媒体文件
├── 临时文件/           # 临时工作文件
├── AI模型/             # AI模型相关资料
├── 测试数据/           # 测试数据
├── 训练数据/           # 训练数据
├── API文档/            # API接口文档
├── 参考资料/           # 参考技术资料
├── 标准规范/           # 相关标准规范
├── 硬件文档/           # 硬件相关文档
└── 系统设计/           # 系统设计文档
```

### 通用目录结构

- `01-architecture/` - 系统总体架构、模块边界、接口关系、部署结构、网络拓扑
- `02-hardware/` - 工控机、相机、线激光、伺服、电源、传感器、接线与安装结构
- `03-software/` - 软件模块设计、采集程序、同步逻辑、处理流程、平台服务
- `04-algorithm/` - 缺陷检测、几何参数计算、同步诊断、帧率节拍等算法
- `05-data-transfer/` - 本地存储、断点续传、4G/5G上传、云端传输方案
- `06-acceptance/` - 验收指标、测试项、试运行记录、交付清单

## 其他目录

- `02-docx/` - Word文档（草稿、定稿、归档）
- `03-scripts/` - 脚本工具（Python/PowerShell/Node）
- `04-analysis/` - 分析过程资料与研究性输出
- `05-temp/` - 临时文件、中间产物
- `06-skills/` - 本地技能与扩展资料
- `99-archive/` - 低频历史归档资料
