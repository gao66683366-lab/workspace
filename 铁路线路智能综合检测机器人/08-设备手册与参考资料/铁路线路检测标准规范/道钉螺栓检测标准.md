## 二、TB/T 564-1992 螺旋道钉

### 2.1 适用范围
本标准规定了螺旋道钉的技术要求、试验方法、检验规则等。
适用于铁路轨道用的螺旋道钉。

### 2.2 检测项目

| 检测项目 | 说明 |
|---------|------|
| 外观检查 | 表面不应有过烧、裂纹、凹痕、毛刺等缺陷 |
| 螺纹检查 | 螺纹表面应光洁，无影响使用的缺陷 |
| 冷弯性能 | 道钉抗弯能力 |
| 拉力载荷 | 道钉抗拉能力 |
| 防锈性能 | 耐腐蚀能力 |
| 螺旋道钉抗拔力 | 抗拔出力 |

## 三、道钉/螺栓状态检测内容

### 3.1 本机器人检测能力

本机器人通过CAM3/CAM4（2D工业相机）检测道钉/螺栓状态：

| 检测内容 | 实现方式 | 说明 |
|---------|----------|------|
| 螺栓缺失 | 视觉检测 | 图像中无螺栓区域 |
| 螺栓松动 | 视觉检测（图像对比） | 螺栓位置偏移 |
| 螺栓断裂 | 视觉检测 | 螺栓断裂特征 |
| 螺栓锈蚀 | 视觉检测 | 锈蚀特征 |
| 紧固状态 | 视觉检测 | 螺栓是否松动 |

### 3.2 检测原理

```
相机采集 → 图像预处理 → 目标检测(YOLO) → 螺栓状态判定
                              ↓
                         完好/缺失/松动/断裂
```

## 四、道钉螺栓常见缺陷

### 4.1 缺陷类型

| 缺陷类型 | 特征描述 | 危害等级 |
|---------|----------|----------|
| **缺失** | 螺栓区域无螺栓图像 | 高 |
| **松动** | 螺栓相对原位置偏移 | 高 |
| **断裂** | 螺栓断裂，图像不完整 | 高 |
| **锈蚀** | 螺栓表面锈斑、变色 | 中 |
| **歪斜** | 螺栓安装角度异常 | 中 |
| **过紧** | 螺栓变形、裂纹 | 中 |

### 4.2 缺陷检测指标

| 检测指标 | 要求 |
|---------|------|
| 螺栓缺失检出率 | ≥95% |
| 螺栓松动检测精度 | ±2mm |
| 锈蚀检测准确率 | ≥90% |
| 单帧检测耗时 | <100ms |

## 六、项目对标情况

### 6.1 检测能力对照

| 检测项目 | 国标要求 | 本机器人能力 | 对标状态 |
|---------|---------|-------------|---------|
| 道钉外观检测 | 外观无缺陷 | CAM3/CAM4视觉检测 | ✅ 可实现 |
| 螺栓缺失检测 | 缺失检出 | YOLO目标检测 | ✅ 可实现 |
| 螺栓松动检测 | 位置偏移检测 | 图像对比分析 | ✅ 可实现 |
| 螺栓断裂检测 | 断裂特征检测 | 视觉检测 | ✅ 可实现 |
| 锈蚀检测 | 锈蚀判定 | 视觉检测 | ✅ 可实现 |

| 指标 | 设计目标 |
|------|---------|
| 螺栓检测精度 | ±1mm（位置） |
| 缺失检出率 | ≥98% |
| 松动检出率 | ≥95% |
| 单螺栓检测耗时 | <50ms |

## 八、备注

1. 道钉/螺栓检测以视觉检测为主，无国家级强制性检测标准
2. 检测指标可参考各产品标准和施工验收标准
3. 本机器人通过图像对比和YOLO深度学习实现螺栓状态检测
4. 发现异常后建议人工复检确认
