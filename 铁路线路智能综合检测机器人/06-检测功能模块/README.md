# 06-检测功能模块

> **定位说明：** 检测功能模块是智能综合检测的具体落地——9大检测功能共享同一套硬件和软件平台，数据同源、同时、同一标准输出。本目录每项检测功能均包含：检测原理、硬件支撑、数据链路、算法处理流程、输出格式、判定标准六个维度，统属于"综合检测"框架，而非孤立的功能罗列。

## 二、检测功能分项说明

### 模块01：轨面缺陷检测

**检测内容：** 检测钢轨轨面裂纹、剥落、擦伤、鱼鳞纹、掉块等表面异常

**硬件支撑：**
- CAM1（覆盖左轨轨面）
- CAM2（覆盖右轨轨面）
- 专用光源（恒定照明，消除环境光干扰）

**数据链路：**

```
相机采集（连续20fps） → 图像预处理 → AI缺陷检测网络 → 缺陷类型+位置+像素掩码
      │
      └── Frame ID + 里程标记 ──▶ 多维融合分析
```

**算法处理：**
- AI目标检测：YOLOv8，输出缺陷类别（裂纹/掉块/凹陷/腐蚀等）
- 像素级分割：UNet，输出缺陷区域像素掩码
- 置信度过滤：低于阈值 → 标记为"疑似"，不直接报警

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 缺陷类型 | 裂纹/掉块/凹陷/腐蚀等 |
| 缺陷等级 | 一级/二级/三级 |
| 里程位置 | 里程值（m）+ Frame ID |
| 像素坐标 | 左上/右下坐标 + 中心点 |
| 置信度 | 0~1 |
| 图像路径 | 原始图像存储路径 |

---

### 模块02：道钉/螺栓检测

**检测内容：** 检测轨枕螺栓/道钉的缺失、松动、歪斜、断裂等状态

**硬件支撑：**
- CAM3（覆盖左轨螺栓区域，25fps，里程触发）
- CAM4（覆盖右轨螺栓区域，25fps，里程触发）
- 补充复核：CAM5/CAM6对重点螺栓位二次采集

**螺栓节距触发策略：**

```
里程脉冲（节距700mm） → 到达螺栓位置 → 触发CAM3/CAM4采集
      │
      └── 螺栓区域ROI自动裁切 → 专注螺栓区域分析
```

**算法处理：**
- 目标检测：YOLOv8，检测螺栓是否存在
- 位置偏差分析：完好螺栓有标准位置，歪斜时相对位置偏移
- 松动判定：结合振动传感器数据（若配置）

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 螺栓编号 | 左轨/右轨 + 序号 |
| 状态 | 完好/缺失/松动/歪斜 |
| 等级 | 一级（二级/三级） |
| 里程位置 | 里程值 |
| 偏移量 | 相对标准位置的像素偏移（歪斜时填写） |

---

### 模块03：3D钢轨轮廓检测

**检测内容：** 检测左右钢轨轨头廓形，计算磨耗量、横移量，与标准廓形比对

**硬件支撑：**
- 激光1（垂直朝下，检测左轨轨头）
- 激光2（垂直朝下，检测右轨轨头）
- 工控机点云处理模块

**三角测量原理：**

```
激光器 ──▶ 投射线性光条到钢轨表面
     │
     ▼
相机（斜置角度）──▶ 拍摄光条形变
     │
     ▼
三角测量计算 ──▶ 轨头三维坐标点序列
```

**廓形比对流程：**

```
点云帧（~1000点） → 滤波去噪 → 廓形平滑 → 标准廓形模板匹配
      │
      ▼
计算磨耗量（最大偏差）→ 横移量（中心偏移）→ 超限报警
```

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 里程位置 | 里程值 |
| 左轨磨耗量 | mm |
| 右轨磨耗量 | mm |
| 左轨横移量 | mm |
| 右轨横移量 | mm |
| 超限标记 | 是/否 |
| 廓形原始点云 | 文件路径 |

---

### 模块04：钢轨波磨检测

**检测内容：** 检测钢轨轨面波浪磨耗的波长、波深、分布里程

**硬件支撑：**
- 3D线激光（连续扫描，获取轨面高程数据）
- CAM5/CAM6（辅助复核）

**波磨分析算法：**

```
3D激光扫描 → 沿轨道方向高程序列 → 频域分析（FFT）
      │
      ▼
提取主频率 → 换算波长 → 计算波深
      │
      ▼
对照《铁路线路检测标准规范》→ 波磨等级判定
```

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 里程范围 | 起始～终止里程 |
| 主波长 | mm |
| 波深 | mm |
| 波磨等级 | 一级/二级/三级 |
| 分布长度 | m |

---

### 模块05：钢轨焊缝检测

**检测内容：** 检测钢轨焊缝区域的缺陷（未焊透、气孔、夹渣等）

**硬件支撑：**
- CAM5/CAM6（补充复核区域，焊缝通常在固定里程）

**算法处理：**
- 图像识别：YOLOv8，检测焊缝区域的异常特征
- 纹理分析：LBP/灰度共生矩阵，量化焊缝区域纹理均匀性

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 焊缝里程 | 里程值 |
| 缺陷类型 | 未焊透/气孔/夹渣/凹凸不平 |
| 等级 | 一级/二级/三级 |
| 图像路径 | 原始图像路径 |

---

### 模块06：轨距检测

**检测内容：** 测量左右钢轨之间的距离，与标准轨距（1435mm）比对

**硬件支撑：**
- 3D线激光点云数据（左右轨轨顶面中心点计算）
- 或专用测距传感器

**计算方法：**

```
左轨轨顶面中心点（激光点云） + 右轨轨顶面中心点 → 计算水平距离
      │
      ▼
  轨距值 = |左轨中心 - 右轨中心|
      │
      ▼
  轨距偏差 = 轨距值 - 标准轨距(1435mm)
```

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 里程位置 | 里程值 |
| 轨距测量值 | mm |
| 轨距偏差 | mm（正/负） |
| 超限标记 | 是/否 |

---

### 模块07：钢轨水平检测（水平度）

**检测内容：** 检测左右钢轨的高差（水平度），判断是否存在水平不平顺

**硬件支撑：**
- 双陀螺仪（姿态参数传感器）

**算法处理：**

```
双陀螺仪角速度积分 → 俯仰角计算 → 左右轨高差 = 俯仰角 × 轨距
      │
      ▼
  高差值（mm）→ 对照标准限值 → 水平等级判定
```

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 里程位置 | 里程值 |
| 水平偏差值 | mm |
| 等级 | 一级/二级/三级 |
| 超限标记 | 是/否 |

---

### 模块08：钢轨高低检测

**检测内容：** 检测沿线路方向的垂直不平顺（高低不平顺）

**硬件支撑：**
- 双陀螺仪（连续采集）

**算法处理：**

```
陀螺仪角速度积分 → 姿态角序列 → 高低不平顺值
      │
      ▼
  与前序区段差值 → 计算变化率 → 高低等级判定
```

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 里程位置 | 里程值 |
| 高低偏差值 | mm |
| 等级 | 一级/二级/三级 |
| 超限标记 | 是/否 |

---

### 模块09：钢轨轨向检测

**检测内容：** 检测沿线路方向的水平偏差（轨向不平顺）

**硬件支撑：**
- 双陀螺仪（连续采集）

**算法处理：**

```
      │
      ▼
  与标准值比对 → 轨向等级判定
```

**输出格式：**

| 字段 | 说明 |
|-----|------|
| 里程位置 | 里程值 |
| 轨向偏差值 | mm |
| 等级 | 一级/二级/三级 |
| 超限标记 | 是/否 |

## 三、综合检测数据融合机制

**核心：所有9个检测模块共用同一套 Frame ID + 里程体系**

```
Frame ID ──▶ 里程 ──▶ 时间戳 ──▶ 9项检测结果
                         │
                         ▼
                  多维融合分析引擎
                         │
                         ▼
              ┌─────────────────────────┐
              │  缺陷A：2D图像裂纹      │
              │  缺陷A：3D廓形无变形    │ ──▶ 降级（疑似）
              │  缺陷A：同位置姿态正常   │
              ├─────────────────────────┤
              │  缺陷B：2D图像掉块      │
              │  缺陷B：3D廓形凹陷     │ ──▶ 升一级（严重）
              │  缺陷B：同位置轨向偏   │
              └─────────────────────────┘
```

> **融合价值：** 综合检测不是9个功能简单并列，而是在统一时空框架下，对同一位点的多维数据进行联合判读，大幅提升判级准确性。
