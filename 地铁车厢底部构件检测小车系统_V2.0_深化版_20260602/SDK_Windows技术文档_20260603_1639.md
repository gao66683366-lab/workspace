## 一、项目概述

RobotLinkSDK是列车底部构件检测机器人触摸屏控制端软件的核心通信中间件，基于C#/.NET 8.0开发，封装了WiFi6 TCP通信、RS-485 Modbus传感器读取、RTSP视频流接收、IO控制四大功能模块。

### 1.1 技术栈

| 层次 | 技术 |
|---|---|
| 框架 | .NET 8.0 / C# |
| 通信 | TCP Socket / SerialPort / RTSP |
| 协议 | STX(0x02) + CMD + LEN + Payload + CRC16 + ETX |
| 校验 | CRC16-Modbus |

### 1.2 解决方案结构

```
RobotInspection.sln
├── RobotLinkSDK/          # SDK类库
│   ├── RobotClient.cs     # 机器人控制客户端
│   ├── SensorReceiver.cs  # 传感器接收器
│   ├── VideoStreamManager.cs  # 视频流管理器
│   ├── IOController.cs    # IO控制器
│   ├── Protocol.cs       # 协议定义
│   ├── DataStructures.cs # 数据模型
│   └── CRC16.cs          # CRC校验
└── RobotTouchUI/          # WPF触摸应用
```

## 二、协议定义

### 2.1 帧格式

```
+--------+--------+--------+--------+--------+--------+--------+
| STX    | CMD    | LEN_L  | LEN_H  | Payload(N) | CRC16_L | CRC16_H | ETX    |
+--------+--------+--------+--------+--------+--------+--------+
| 0x02   | 1B     | 1B     | 1B     | N字节      | 1B      | 1B      | 0x03   |
+--------+--------+--------+--------+--------+--------+--------+
```

- **STX**: 帧起始符 (0x02)
- **CMD**: 命令码 (1字节)
- **LEN**: 载荷长度 (2字节, little-endian)
- **Payload**: 载荷数据 (N字节)
- **CRC16**: CRC校验码 (覆盖STX~Payload)
- **ETX**: 帧结束符 (0x03)

### 2.2 端口配置

| 通道 | 端口 | 协议 | 说明 |
|---|---|---|---|
| TCP命令 | 8001 | TCP | 控制指令+应答 |
| TCP遥测 | 8002 | TCP | 实时数据推送 |
| RTSP视频 | 554 | RTSP | H.264视频流 |

### 2.3 命令码定义（20条）

| 命令码 | 名称 | 说明 |
|---|---|---|
| 0x01 | CMD_HEARTBEAT | 心跳 |
| 0x10 | CMD_MOTION_ABS | 绝对运动 |

## 三、RobotClient.cs

机器人控制客户端，负责WiFi6 TCP通信、命令发送、心跳维持、连接管理。

### 3.1 核心特性

- 双通道TCP连接（命令8001 + 遥测8002）
- 异步命令发送（带超时重发）
- 心跳保活（2秒间隔，3次失败触发重连）
- 线程安全设计
- 事件驱动架构

### 3.2 主要方法

#### 连接管理

```csharp
// 异步连接
Task<bool> ConnectAsync(string ip, int commandPort = 8001, int telemetryPort = 8002)

// 同步连接
bool Connect(string ip, int commandPort = 8001, int telemetryPort = 8002)

// 断开连接
void Disconnect()
```

#### 运动控制

```csharp
Task SetSpeedAsync(float speed)       // 设置速度 (0~0.5m/s)
Task MoveAbsAsync(float distance)     // 绝对运动
Task MoveRelAsync(float distance)      // 相对运动
Task MoveStopAsync()                  // 停止
Task MoveHomeAsync()                  // 回零
```

#### 云台控制

```csharp
Task PTZAngleAsync(byte camera, short pan, short tilt)  // 角度控制
Task PTZPresetAsync(byte camera, byte presetId)          // 预置位
Task PTZResetAsync(byte camera)                          // 复位
```

#### 相机控制

```csharp
Task CameraZoomAsync(byte camera, ushort zoom)    // 变倍
Task CameraFocusAsync(byte camera, ushort focus)  // 聚焦
Task CameraCaptureAsync(byte camera)              // 抓拍
```

#### 光源控制

```csharp
Task SetBrightnessAsync(byte brightness)  // 亮度 (0~100%)
Task SetLightSwitchAsync(byte lightId, bool onoff)  // 开关
```

#### IO控制

```csharp
Task SetDOAsync(byte index, bool value)  // 数字输出
Task GetDIAsync()                         // 读取数字输入
Task SetPWMAsync(byte channel, byte dutyCycle)  // PWM输出
```

#### 系统控制

```csharp
Task RequestStatusAsync()   // 请求系统状态
Task SyncTimeAsync()        // 时间同步
Task HeartbeatAsync()       // 发送心跳
```

### 3.3 事件

```csharp
event EventHandler<bool>? OnConnectionChanged      // 连接状态变化
event EventHandler<SystemStatusData>? OnSystemStatus  // 系统状态
event EventHandler<AlarmEventArgs>? OnAlarm        // 报警事件
event EventHandler<TelemetryData>? OnTelemetry    // 遥测数据
```

### 3.4 使用示例

```csharp
var robot = new RobotClient();
robot.OnConnectionChanged += (s, connected) => 
    Console.WriteLine($"连接状态: {(connected ? "已连接" : "断开")}");
robot.OnSystemStatus += (s, status) => 
    Console.WriteLine($"模式: {status.Mode}, 错误: {status.ErrorCode}");

await robot.ConnectAsync("192.168.1.100");
await robot.SetSpeedAsync(0.3f);
await robot.MoveForwardAsync();
await robot.PTZAngleAsync(0, 45, -30);
await robot.SetBrightnessAsync(80);
await robot.MoveStopAsync();
robot.Disconnect();
```

## 四、SensorReceiver.cs

传感器接收器，通过RS-485 Modbus RTU总线轮询传感器数据。

### 4.1 总线参数

| 参数 | 值 |
|---|---|
| 波特率 | 9600-115200bps |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验 | 无 |
| 模式 | 半双工主从 |

### 4.2 支持的传感器

| 从机地址 | 传感器类型 | 寄存器数量 |
|---|---|---|
| 0x01 | IMU惯性测量单元 | 12寄存器 |
| 0x02 | 测距传感器 | 4寄存器 |
| 0x03 | BMS电池管理 | 10寄存器 |

### 4.3 主要方法

```csharp
// 构造函数
SensorReceiver(string portName = "COM1", int baudRate = 115200)

// 启动/停止
void Start()
void Stop()
void SetPollInterval(int ms)

// 属性
IMUData? LatestIMU { get; }
RangingData? LatestRanging { get; }
BMSData? LatestBMS { get; }
```

### 4.4 事件

```csharp
event EventHandler<IMUData>? OnIMU       // IMU数据回调 (20Hz)
event EventHandler<RangingData>? OnRanging  // 测距回调 (20Hz)
event EventHandler<BMSData>? OnBMS       // BMS回调 (1Hz)
```

### 4.5 使用示例

```csharp
var sensor = new SensorReceiver("COM1", 115200);
sensor.OnIMU += (s, imu) => 
    Console.WriteLine($"Roll:{imu.Roll:F1}° Pitch:{imu.Pitch:F1}° Yaw:{imu.Yaw:F1}°");
sensor.OnRanging += (s, r) => 
    Console.WriteLine($"前:{r.Front:F2}m 后:{r.Rear:F2}m");
sensor.OnBMS += (s, b) => 
    Console.WriteLine($"{b.Voltage:F1}V {b.SOC}% {b.TempMax}℃");
sensor.Start();
```

## 五、VideoStreamManager.cs

视频流管理器，通过RTSP协议接收H.264视频流，集成FFmpeg解码。

### 5.1 参数

| 参数 | 值 |
|---|---|
| 端口 | 554 |
| 编码 | H.264 |
| 协议 | RTSP over TCP |
| 延迟 | 可配置 (默认0ms) |

### 5.2 主要方法

```csharp
// 连接视频流
void Connect(string frontUrl, string rearUrl, int latencyMs = 0)

// 异步连接（带超时）
Task ConnectAsync(string frontUrl, string rearUrl, int latencyMs = 0, int timeoutMs = 5000)

// 断开连接
void Disconnect()
```

### 5.3 事件

```csharp
event EventHandler<byte[]>? OnFrontFrame  // 前相机帧回调 (NV12格式)
event EventHandler<byte[]>? OnRearFrame   // 后相机帧回调 (NV12格式)
```

### 5.4 使用示例

```csharp
var video = new VideoStreamManager();
video.OnFrontFrame += (s, frame) => 
{
    // frame为NV12格式原始图像数据
    // 需要转换为RGB/BGR显示
};
video.Connect(
    "rtsp://192.168.1.100:554/stream/front?latency=0",
    "rtsp://192.168.1.100:554/stream/rear?latency=0"
);
```

## 六、IOController.cs

IO控制器，管理8路数字输入、8路数字输出、4路PWM调光。

### 6.1 主要方法

```csharp
// 构造函数
IOController(string portName = "COM2", int baudRate = 9600)

// 启动/停止监控
void Start()
void Stop()
void SetPollInterval(int ms)

// 数字输出控制
void SetDO(byte index, bool value)           // 设置单个DO
void SetDO(byte mask, byte values)           // 批量设置DO

// 数字输入读取
byte GetDI()                                  // 读取所有DI
bool GetDI(byte index)                        // 读取单个DI

// PWM控制
void SetPWM(byte channel, byte dutyCycle)     // 设置单路PWM
void SetPWM(byte duty0, byte duty1, byte duty2, byte duty3)  // 设置4路PWM
```

### 6.2 事件

```csharp
event EventHandler<DIChangedEventArgs>? OnDIChanged  // DI状态变化
```

### 6.3 使用示例

```csharp
var io = new IOController("COM2", 9600);
io.OnDIChanged += (s, e) => 
    Console.WriteLine($"DI变化: 0x{e.ChangedBits:X2}, 当前: 0x{e.DIStatus:X2}");

io.Start();
io.SetDO(0, true);        // DO0输出高
io.SetPWM(0, 80);         // PWM0占空比80%
bool di0 = io.GetDI(0);   // 读取DI0状态
io.SetDO(0, false);       // DO0输出低
```

### 10.1 CRC16-Modbus校验算法

```csharp
public static ushort Calc(byte[] data, int offset, int length)
{
    ushort crc = 0xFFFF;
    for (int i = offset; i < offset + length; i++)
    {
        crc = (ushort)((crc << 8) ^ Table[((crc >> 8) ^ data[i]) & 0xFF]);
    }
    return crc;
}
```
