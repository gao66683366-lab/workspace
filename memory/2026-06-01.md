# 2026-06-01 日志

## 列车底部构件检测机器人 — 技术方案 V1.0 确认

### 项目背景
- 与"铁路线路智能检测机器人"（检测线路）是不同项目
- 检测对象：列车底部构件（车底构架、转向架部件）
- 场景：入库静态检测或库线动态检测

### 机器人形态：轨底行走式
- 机器人在**钢轨轨底间隙内行走**（不是骑在轨道上）
- 四轮驱动：轨底驱动轮×4 + 枕木驱动轮×4
- 两种模式：
  - 轨底模式：电缸收回，轨底驱动轮着轨 → 进入股道检测
  - 枕木模式：电缸伸出，枕木驱动轮着枕木 → 跨股道转移
- 升降电缸控制模式切换，含避夹板保护逻辑

### 检测方式：双云台仰拍
| 云台 | 相机 | 角度 | 功能 |
|---|---|---|---|
| 前云台 | 4K工业相机（全局快门） | 仰拍 +45°~+90° | 拍摄车底底架（构架/横梁/减震器/液压元件） |
| 后云台 | 1080P工业相机 | 斜向后 | 监控轨道行走安全 |

- 前云台精度 <0.1°（高精度数字舵机）
- 后云台精度 <0.5°（标准RC舵机）
- 车底净空 0.5–1.2m，前云台需支持近乎垂直向上

### 8项检测功能（V1.0）
| # | 检测功能 | 说明 |
|---|---|---|
| 01 | 转向架构件缺陷检测 | 裂纹、变形、腐蚀 |
| 02 | 螺栓松紧检测 | 缺失、松动、歪斜 |
| 03 | 减震器状态检测 | 漏油、变形、裂纹 |
| 04 | 制动系统检测 | 闸瓦磨耗、螺栓异常 |
| 05 | 齿轮箱异常检测 | 油渍、裂纹、外壳 |
| 06 | 牵引电机检测 | 接线盒、裂纹 |
| 07 | 轨枕/横梁裂纹检测 | 底部结构裂纹 |
| 08 | 异物入侵检测 | 遗留物 |

- 5G/WiFi6 传输网：远程监控、数据上传
