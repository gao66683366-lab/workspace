> 本文件收录钢轨廓形、磨耗检测相关的标准规范及技术指标，用于指导2×3D线激光轨头轮廓检测系统的对标工作。

## 二、TB/T 2097-1989 钢轨允许磨耗限度

### 2.1 适用范围
本标准规定了钢轨的垂直磨耗、侧面磨耗及波形磨耗的允许限度。
适用于38、43、50及60kg/m国产与非国产钢轨。

### 2.2 磨耗类型定义

| 磨耗类型 | 定义 | 测量位置 |
|---------|------|----------|
| **垂直磨耗** | 轨顶面中心的磨耗 | 轨顶中心 |
| **侧面磨耗** | 轨距角的磨耗 | 轨距角处 |
| **波形磨耗** | 轨顶面的波浪形磨耗 | 波长20-200mm范围 |
| **总磨耗** | 垂直磨耗 + 侧面磨耗 | 叠加计算 |

### 2.3 磨耗限度参考值

> 以下数据为常见限值参考，实际应以正式标准文本为准

| 钢轨类型 | 垂直磨耗限值(mm) | 侧面磨耗限值(mm) | 总磨耗限值(mm) |
|---------|-----------------|------------------|---------------|
| 60 kg/m | 8-10 | 8-10 | 12-14 |
| 50 kg/m | 7-9 | 7-9 | 10-12 |
| 43 kg/m | 6-8 | 6-8 | 8-10 |

**注**：重伤标准通常为垂直磨耗>10mm或侧面磨耗>10mm或总磨耗>14mm（60kg/m钢轨）

## 三、T/CCTAS 179-2024 钢轨廓形打磨验收规范

### 3.1 适用范围
本文件适用于高速铁路、普速铁路（含重载铁路）、城际铁路钢轨廓形打磨的验收。

### 3.2 验收内容
1. 钢轨廓形打磨质量验收
2. 允许偏差和技术要求

### 3.3 钢轨接触光带要求

| 铁路类型 | 光带宽度要求 | 光带位置要求 |
|----------|-------------|-------------|
| 高速铁路 | 20mm～30mm | 居中或居中偏内 |
| 普速铁路 | 25mm～35mm | 居中或居中偏内 |

**测量时机**：钢轨接触光带未形成前，宜在钢轨上喷漆，车辆通过形成光带后，7日内对钢轨接触光带进行测量。

## 四、3D线激光钢轨廓形检测技术指标

### 4.1 检测精度要求

| 检测项目 | 精度要求 | 说明 |
|---------|---------|------|
| 垂直磨耗测量 | ±0.2mm | 满足铁路维修标准 |
| 侧面磨耗测量 | ±0.2mm | 满足铁路维修标准 |
| 重复测量误差 | <0.1mm | 绝对测量精度 |
| 绝对测量精度 | 最高可达0.014mm | 研究论文数据 |
| 高度测量MAD | 0.009mm | 研究论文数据 |
| 横向测量MAD | 0.039mm | 研究论文数据 |
| 高度测量RMSE | 0.011mm | 研究论文数据 |
| 横向测量RMSE | 0.048mm | 研究论文数据 |

### 4.2 测量系统指标（参考商用产品SPM）

| 参数 | 指标 |
|------|------|
| 重复精度 | ≤0.005mm |
| 测量精度 | ≤0.01mm |
| 适用钢轨类型 | 43～75kg/m |
| 测量内容 | 垂直磨耗、侧面磨耗、等效锥度、轮轨接触特性 |
| 测量耗时 | 不超过5秒 |
| 整机重量 | <1.0kg |

### 4.3 本项目2×3D线激光系统定位

本机器人配置：
- **2个3D线激光**：左右轨头轮廓检测（垂直朝下）
- **检测目标**：轨头廓形、磨耗测量、轮廓偏差计算

## 五、钢轨廓形检测方法

### 5.1 基本原理

```
线结构光垂直"切割"钢轨 → 摄像传感器获取钢轨断面轮廓图像 
→ 实时处理 → 钢轨轮廓及磨耗信息
```

### 5.2 检测流程

```
1. 激光线照射钢轨横截面
2. 相机采集激光线轮廓
3. 点云重建钢轨断面
4. 与标准廓形对比
5. 计算垂直磨耗、侧面磨耗
6. 输出磨耗值及报警
```

### 5.3 标准钢轨廓形参数

- **轨底坡**：1:40
- **轨顶面宽度**：约70-80mm（60kg/m钢轨轨顶宽约73mm）
- **轨头高度**：约90mm（60kg/m钢轨）
- **标准轮廓坐标系**：基于轨底中心线

## 六、项目对标情况

### 6.1 检测能力对照

| 检测项目 | 国标要求 | 本机器人能力 | 对标状态 |
|---------|---------|-------------|---------|
| 轨头廓形 | 需满足±0.2mm精度 | 2×3D线激光（垂直朝下） | ✅ 可实现 |
| 垂直磨耗 | 允许磨耗限度内 | 计算轨顶中心偏差 | ✅ 可实现 |
| 侧面磨耗 | 允许磨耗限度内 | 计算轨距角处偏差 | ✅ 可实现 |
| 波磨检测 | 波长20-200mm | 轮廓数据可分析 | ✅ 可实现 |

### 6.2 精度设计目标

| 指标 | 设计目标 | 参考依据 |
|------|---------|---------|
| 廓形测量精度 | ±0.1mm | 行业高水平要求 |
| 磨耗计算精度 | ±0.2mm | TB/T 2097-1989要求 |
| 重复性 | <0.1mm | 商用检测仪水平 |
