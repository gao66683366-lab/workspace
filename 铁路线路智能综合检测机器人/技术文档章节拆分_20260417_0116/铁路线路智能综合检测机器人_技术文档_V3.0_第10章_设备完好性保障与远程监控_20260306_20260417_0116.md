## 10 设备完好性保障与远程监控

### 一、设备运行完好性状态监测

在检测过程中，对机器人上所有检测设备进行实时完好性监控，确保设备始终处于良好工作状态。设备安全稳定运行是检测数据质量的根基。

```
┌─────────────────────────────────────────────────┐
│           检测设备完好性实时监控                      │
│                                                    │
│  6路工业相机 ──→ 在线状态 / 帧率 / 丢包率 / 温度 │
│  2路3D激光 ──→ 功率 / 数据率 / 温度             │
│  2路陀螺仪 ──→ 角速度 / 姿态角 / 健康状态        │
│  工控机本体 ──→ CPU / 内存 / 磁盘 / 进程 / 网络  │
│  工业总线 ────→ 通信状态 / 故障码               │
│  通信链路 ────→ 移动网络信号 / 带宽 / 延迟        │
└─────────────────────────────────────────────────┘
```

**完好性监控详细指标：**

| 设备 | 监控参数 | 正常范围 | 告警阈值 | 处理动作 |
|---|---|---|---|---|
| 工业相机×6 | 帧率 | 25fps | < 20fps 持续 5s | 标记该相机数据缺失，触发补拍逻辑 |
| | 丢包率 | < 0.1% | > 1% 持续 10s | 降权使用，任务后检修 |
| | 相机温度 | 20~50°C | > 55°C | 降频，> 60°C 触发急停 |
| | 在线状态 | 心跳正常 | 连续 3 次无心跳 | 标记离线，启动冗余逻辑 |
| 3D激光×2 | 数据率 | 100Hz 稳定 | < 95Hz 持续 30s | 标记数据异常，降级运行 |
| | 激光功率 | 额定功率±5% | 超出±10% | 自动降功率保护 |
| | 温度 | 20~45°C | > 50°C | 降频，> 55°C 停机冷却 |
| 双陀螺仪 | 角速度漂移 | < 0.01°/s | > 0.05°/s | 触发陀螺仪重置 |
| | 姿态角精度 | ±0.01° | 超出±0.05° | 切换备用陀螺仪 |
| | 两轴数据一致性 | 差值<0.5° | 差值>1° | 标记异常，人工检修 |
| 工控机 | CPU占用 | < 60% | > 80% 持续 1min | 触发GC，降AI推理并发 |
| | 内存占用 | < 50% | > 85% | 清理历史缓存 |
| | 磁盘剩余空间 | > 20% | < 10% | 删除最旧任务数据 |
| | GPU温度 | 30~70°C | > 80°C | 降GPU频率 |
| | 进程存活 | 全部正常 | 任一进程退出 | 自动重启该服务 |
| 工业总线伺服运控系统 | 通信状态 | 心跳正常 | 连续 3 次中断 | 触发急停，保护电机 |
| | 伺服驱动故障码 | 0 | 非0 | 立即告警，详细记录故障码 |
| | 电机温度 | < 80°C | > 80°C | 降载保护，> 90°C 急停 |
| | 运行速度一致性 | 速度偏差 < ±5% | 偏差 > ±10% | 触发定速巡航重校正 |
| | 急停回路状态 | 正常 | 回路断开 | 立即停车，导向安全侧 |
| 测距传感器矩阵 | 传感器在线状态 | 心跳正常 | 连续 3 次无心跳 | 标记该传感器离线 |
| | 测量数据率 | 100Hz 稳定 | < 95Hz 持续 10s | 标记数据异常 |
| | 测量精度 | ±1mm | 超出 ±3mm | 触发重校正或更换 |
| | 传感器温度 | 20~50°C | > 55°C | 降频，> 60°C 停机 |
| 移动网络 | 信号强度 | > -85dBm | < -100dBm | 降数据传输优先级 |
| | 传输延迟 | < 200ms | > 500ms | 切换本地存储模式 |
| | 带宽 | > 5Mbps | < 1Mbps | 暂停非紧急数据上传 |

**设备故障自检机制：**

| 自检类型 | 频率 | 内容 |
|---|---|---|
| 传感器自检 | 每次任务前 | 相机曝光校正 / 激光功率校准 / 陀螺仪零偏校准 |
| 帧率自检 | 运行时持续 | 统计每分钟实际帧率，与设定值对比 |
| 数据完整性自检 | 运行时持续 | 统计应收帧数 vs 实收帧数 |
| 通信自检 | 每 10s | 各节点心跳包检测，标记超时节点 |

**冗余备份策略：**
- 6路相机中任意1~2路故障时，其余相机可覆盖检测，任务不中断（标注数据缺失区段）
- 双陀螺仪数据互为备份，单轴异常时自动切换
- 工控机关键进程双重部署，任一异常自动切换备用进程

### 二、智能安全供电管理

用电安全管理是保障机器人安全运行的核心，贯穿检测任务全流程。

本系统的智能安全供电管理体系覆盖所有用电节点，实现从蓄电池到各传感器设备的全链路电流、电压、温度实时监控，确保设备在检测过程中不会因电力问题而受损或停机。

**蓄电池安全管理 — 全参数实时监控**

蓄电池是检测机器人的"心脏"，本系统对蓄电池实施全参数实时监控：

| 监控参数 | 说明 | 安全机制 |
|---|---|---|
| **电压** | 实时监控蓄电池组电压（总电压 + 单体电压） | 低于阈值立即报警并触发返航，防止过放 |
| **电流** | 充放电电流实时监测 | 充电过流自动断充，放电异常预警 |
| **SOC 电量** | 剩余电量百分比实时显示（精度±2%） | 电量低于预设阈值前主动报警 |
| **SOH 健康度** | 电池健康状态评估（当前容量/初始容量） | 健康度低于70%时提示更换 |
| **续航预估** | 基于当前电量 + 历史能耗统计 | 实时预估剩余可行驶里程，提前规划返航 |
| **BMS 电路板温度** | 蓄电池管理系统电路板温度实时采集 | 电路板温度超限立即报警，防止BMS失效引发安全事故 |
| **智能电路管理板温度** | 智能电源管理板温度实时采集 | 管理板温度超限自动降载或断电，防止过热损坏 |

**BMS（蓄电池管理系统）详解：**

| BMS 功能 | 说明 |
|---|---|
| 电池均衡 | 单体电池电压均衡，防止过充/过放，延长电池寿命 |
| 短路保护 | 检测到短路电流时，毫秒级切断电池输出 |
| 温度保护 | BMS板温度传感器检测，温度超限立即告警并降载 |
| 通信接口 | CAN总线与工控机实时通信，每秒刷新电池状态 |
| 告警输出 | 独立硬线告警信号，即使软件崩溃也能触发保护 |

**三级保护机制：**

| 级别 | 触发条件 | 动作 |
|---|---|---|
| **预警** | 电量 < 20% 或 电压接近低限 | 云端推送预警，提醒准备返航 |
| **告警** | 电量 < 10% 或 BMS板温度 > 55°C | 声光报警 + 强制触发一键返航 |
| **保护** | 电压/电流严重超限或管理板温度 > 70°C | 硬件独立保护回路立即切断供电，保护设备和线路安全 |

> BMS 电路板和智能电路管理板的温度实时采集，是防止蓄电池管理系统自身失效引发安全事故的最后一道防线，填补了传统监控方案的关键盲区。

**续航里程精准估算算法：**

$$
R_{remaining} = \frac{SOC \times C_{rated} \times \overline{v}}{P_{avg}}
$$

其中：
- $R_{remaining}$：剩余续航里程（km）
- $SOC$：当前电量百分比（0~1）
- $C_{rated}$：电池额定容量（Ah）
- $\overline{v}$：历史平均行驶速度（km/h）
- $P_{avg}$：历史平均功耗（kW）

**检测设备用电管理 — 全节点电流电压实时监控**

整机所有用电节点均纳入实时监控，任何电流电压异常均可快速定位：

| 用电节点 | 监控参数 | 正常电流范围 | 告警阈值 |
|---|---|---|---|
| 6路工业相机 | 每路电流 | 0.5~0.8A @ 12V | > 1.0A 或 < 0.3A |
| 2路3D激光 | 每路电流 | 1.0~2.0A @ 24V | > 2.5A 或 < 0.5A |
| 2路双陀螺仪 | 每路电流 | 0.1~0.3A @ 5V | > 0.5A |
| 工控机 | 总电流 | 2.0~4.0A @ 24V | > 5.0A |
| 工业总线伺服驱动 | 每轴电流 | 0~额定电流 | 超过额定电流持续 5s |
| 测距传感器矩阵 | 每轴电流 | 0.1~0.3A @ 12V | > 0.5A 或 < 0.05A |

> **实时反馈机制**：所有设备的电流、电压数据**每秒刷新**，异常节点可在云端**秒级定位**，大幅缩短故障排查时间。

**运动控制安全保障 — 工业总线硬实时守护**

| 监控项 | 说明 |
|---|---|
| 工业总线通信状态 | 实时监控通信是否中断，CAN FD 心跳每 10ms |
| 伺服驱动状态 | 实时获取驱动故障码，故障时立即记录并告警 |
| 急停回路状态 | 硬件独立急停回路，失效自动导向安全侧（停车） |
| 运行速度一致性 | 定速巡航保障采集数据速度稳定，不受坡度影响 |
| 电机温度 | 实时监控，> 80°C 触发降载保护 |

### 三、远程监测控制能力

本系统具备完整的远程监测与控制能力，支持在云端对现场设备进行全方位的监控和操作：

**远程控制架构：**

```
云端监控中心
     │
     │ 4G/5G 移动网络（MQTT + TLS加密）
     │
     ▼
边缘网关（工控机）
     │
     ├──→ 设备控制指令（CAN总线/串口）
     ├──→ 传感器参数调整（相机/GPU）
     └──→ 系统服务管理（进程启停）
```

**远程控制能力详情：**

| 能力 | 说明 | 响应延迟 |
|---|---|---|
| 实时状态查看 | 云端实时查看任一设备节点的电流/电压/温度/帧率/位置 | < 1s |
| 参数远程调优 | 远程调整相机曝光/帧率/检测阈值，无需到场 | < 3s |
| 设备远程重启 | 单节点故障可远程重启对应服务或重启整个系统 | < 30s |
| 故障隔离 | 单节点故障自动隔离，不影响整体任务执行 | 自动 |
| 远程固件升级 | 传感器固件可通过移动网络远程推送升级包 | 按包大小 |
| 远程日志拉取 | 远程拉取任意设备节点的运行日志 | < 10s |
| 远程指令下发 | 远程下发控制指令（启停/速度/返航） | < 1s |

**远程控制安全机制：**

| 措施 | 说明 |
|---|---|
| 指令认证 | 所有远程指令携带 Token 认证，防伪造 |
| 操作日志 | 所有远程操作记录时间戳、操作者、操作内容 |
| 权限分级 | 不同权限账户可执行的远程操作不同 |
| 指令确认 | 高危指令（急停/复位）需二次确认 |

### 四、四类用户价值

本系统面向四类用户提供差异化的使用价值和操作界面：

| 用户角色 | 使用场景 | 核心价值 | 系统入口 |
|---|---|---|---|
| **现场操作人员** | 出发前检查设备状态；任务中监控进度；异常时急停 | 快速就绪，3步启机；实时掌握任务进度；触摸屏一键操作 | 触摸屏本地端 |
| **远程监控中心** | 多台设备集中监控；大屏GIS可视化；实时缺陷告警 | 全线设备状态和用电数据一目了然；缺陷告警秒级推送 | Web端大屏/PC端 |
| **管理决策者** | 查看设备利用率/故障率/出勤率统计；审批养护计划 | 排班和资源调配有据可依；养护决策有数据支撑 | Web端管理后台 |
| **运维保障团队** | 设备故障提前预警；远程诊断故障根因；远程调参升级 | 减少现场出勤；故障远程可排查；远程升级减少停机时间 | PC端运维平台 |

**现场操作人员典型工作流：**

```
1. 出发前检查（触摸屏）
   └→ 确认6路相机/2路激光/双陀螺仪/BMS状态全部在线
   
2. 任务配置
   └→ 输入本次检测里程区段（如 K100 ~ K110）
   └→ 选择检测速度（默认1m/s）
   
3. 一键出发
   └→ 机器人自动定速巡航，开始检测
   
4. 作业中监控
   └→ 触摸屏实时显示：里程/速度/电量/缺陷数量
   └→ 发现异常可随时按急停或返航
   
5. 任务结束
   └→ 自动返航或手动返航
   └→ 触摸屏提示"任务完成，数据已上传"
```

**运维保障团队远程诊断流程：**

```
故障告警推送（短信/邮件）
    ↓
远程登录运维平台，查看设备状态
    ↓
调取实时参数（电流/电压/温度/日志）
    ↓
判断故障类型：
  - 软件故障 → 远程重启对应服务（< 30s）
  - 参数异常 → 远程调参（< 3s）
  - 硬件故障 → 生成检修工单，安排现场出勤
    ↓
故障解决，记录归档，更新设备健康档案
```
