## 二、TB/T 3355-2023 核心内容

### 2.1 适用范围
- 1435mm标准轨距新建、改建铁路线路验收
- 运营铁路线路评价
- 城市轨道交通可参照使用

### 2.2 检测项目（7项几何参数 + 2项振动）

| 序号 | 检测项目 | 说明 |
|------|---------|------|
| 1 | 高低 | 钢轨顶面竖向不平顺 |
| 2 | 左轨向 | 左股钢轨纵向不平顺 |
| 3 | 右轨向 | 右股钢轨纵向不平顺 |
| 4 | 轨距 | 左右钢轨内侧距离 |
| 5 | 水平 | 左右钢轨顶面高差 |
| 6 | 三角坑 | 水平变化率（基长6.25m） |
| 7 | 复合不平顺 | 轨向与水平的叠加 |
| 8 | 轨距变化率 | 纵向差分 |
| 9 | 车体垂向振动加速度 | - |
| 10 | 车体横向振动加速度 | - |

### 2.3 评价方法

**局部峰值管理（扣分法）**：
| 等级 | 每处扣分 | 处置建议 |
|------|---------|----------|
| Ⅰ级 | 1分 | 需关注 |
| Ⅱ级 | 5分 | 计划维修 |
| Ⅲ级 | 100分 | 立即维修 |
| Ⅳ级 | 301分 | 限速/中断 |

**区段均值管理：TQI轨道质量指数**
- 定义：200m区段内7项几何参数单项标准差之和
- 公式：TQI = Σσᵢ (i=1 to 7)
- 7项：左高低、右高低、左轨向、右轨向、轨距、水平、三角坑
- TQI越小，轨道越平顺

## 五、项目对标情况

### 5.1 检测项目覆盖

| 国标检测项目 | 本机器人能力 | 实现方式 |
|-------------|-------------|----------|
| 高低 | ✅ | 陀螺仪Pitch + 里程积分 |
| 轨向 | ✅ | 陀螺仪Yaw + 里程积分 |
| 轨距 | ✅ | 8个测距传感器横向相加 |
| 水平/超高 | ✅ | 陀螺仪Roll × 轨距基准 |
| 三角坑 | ✅ | 水平变化率（相邻点差分） |
| 轨距变化率 | ✅ | 纵向差分计算 |
| 轨面缺陷 | ✅ | 6×2D工业相机 + YOLO |
| 轨头轮廓 | ✅ | 2×3D线激光 |
| 螺栓/道钉状态 | ✅ | CAM3/CAM4触发采集 |
| 车体振动 | ✅ | 陀螺仪/加速度数据 |

### 5.2 精度要求对标

| 检测指标 | 国标要求（作业验收） | 本机器人设计指标 | 对标状态 |
|---------|---------------------|----------------|---------|
| 轨距 | ±1mm | ±0.5mm（目标） | ✅ 超过 |
| 水平 | ±2mm | ±0.5mm（目标） | ✅ 超过 |
| 高低 | 2mm/10m弦 | ±1mm（目标） | ✅ 超过 |
| 轨向 | 2mm/10m弦 | ±1mm（目标） | ✅ 超过 |
