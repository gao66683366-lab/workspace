# 2. 系统总体设计

## 2.1 设计理念

"实时测距感知 + Stewart并联执行"——以单点测距传感器为水平基准感知层，以六自由度Stewart并联平台为执行层，构建闭环姿态稳定系统。

系统核心思想：轨道无论怎样起伏，通过实时检测轨道高程并驱动Stewart平台反向补偿，使平台始终保持绝对水平。

## 2.4 系统技术参数

| 组件 | 参数 | 说明 |
|---|---|---|
| Stewart平台 | 6-DOF并联机构 | 上平台直径φ200~400mm |
| 电动缸 | 伺服电动缸×6 | 行程20mm，推力≥200N |
| 单点测距传感器 | 红外相位式，2000Hz | 精度±0.15mm，量程50~300mm |
| 姿态传感器 | HWT905，200Hz | 横滚角/俯仰角/偏航角 |
| 主控 | Jetson AGX Orin | 32 TOPS算力 |
| 通信架构 | EtherCAT + 千兆以太网 | 硬实时+高速数传双网隔离 |
| 防护等级 | IP54（工控机）/ IP67（传感器） | 现场级防护 |

## 2.6 系统功能定义

| 功能编号 | 功能名称 | 功能描述 |
|---|---|---|
| F01 | 轨道高程实时检测 | 单点测距传感器2000Hz实时采集轨道表面高程 |
| F02 | 姿态角实时解算 | 基于双传感器差值计算横滚角和俯仰角 |
| F03 | EKF融合滤波 | 融合测距数据与HWT905姿态数据，提高测量精度 |
| F04 | 逆运动学求解 | 根据目标姿态反解6个电动缸的目标长度 |
| F05 | 平台实时调姿 | Stewart平台6缸协同伸缩，补偿轨道起伏 |
| F06 | 远程监控 | 实时姿态数据上传，Web端查看与告警 |
| F07 | 故障自恢复 | 异常情况下自动复位到安全姿态 |
