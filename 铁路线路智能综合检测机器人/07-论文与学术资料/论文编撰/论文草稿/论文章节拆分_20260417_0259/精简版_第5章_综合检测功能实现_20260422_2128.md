# 面向铁路线路的智能视觉与多模态感知融合综合检测系统研究与实现

## 第5章 综合检测功能实现

综合检测的核心价值在于：以一台设备同步完成8项检测功能，所有数据共享同一时空基准，数据天然对齐，为多源融合判定提供了物理基础。

8项检测功能按数据来源可分为两组：**视觉检测组**（功能01~05）以2D工业相机和3D线激光为主要数据源；**几何参数检测组**（功能06~08）以测距传感器矩阵和HWT905姿态传感器为主要数据源。

### 5.1 视觉检测功能群（功能01~05）

视觉检测功能群利用2D工业相机和3D线激光实现轨面状态和几何轮廓的检测，各功能共享感知硬件和融合框架。

**轨面缺陷检测（功能01）** 利用2D工业相机采集轨面图像，AI模型实时识别裂纹、掉块、凹陷等缺陷，输出缺陷类型、位置和置信度。当融合置信度C_fusion低于阈值时，触发人工复核流程。

**道钉/螺栓检测（功能02）** 基于轨面图像利用深度学习模型识别道钉/螺栓的缺失、松动和歪斜状态。道钉标准节距为700mm，算法通过里程序列分析判断状态：若相邻道钉间距Δs满足690mm<Δs<710mm，则判定为正常。

**3D钢轨轮廓检测（功能03）** 利用3D线激光（垂直朝下）扫描轨头轮廓，通过点云数据计算垂直磨耗ΔV和侧面磨耗ΔS，超限时输出告警。

**波磨与焊缝检测（功能04~05）** 波磨检测基于3D点云频谱分析，识别周期性波磨并计算波深；焊缝检测利用2D图像识别焊缝区域缺陷类型和等级。

### 5.2 几何参数检测功能群（功能06~08）

几何参数检测功能群利用测距传感器矩阵和HWT905姿态传感器实现轨距、水平和高低三项几何参数的检测。

**轨距检测（功能06）** 采用测距传感器矩阵直接测量左、右轨内侧距离，测点位于轨顶以下16mm处，与姿态无关，实测轨距G_actual(s) = D_L(s) + D_R(s)，偏差G_deviation(s) = G_actual(s) - 1435mm。

**水平检测（功能07）** 利用HWT905横滚角计算左右轨超高差：Δh = 1435 × sin(θ_r)。

**高低检测（功能08）** 采用单点测距传感器（2000Hz）作为主测手段，HWT905姿态传感器（200Hz）用于俯仰角校正，通过EKF滤波融合得到轨道高程值。EKF状态向量为x_k = [h_track(k), h_veh(k), v_veh(k)]ᵀ，观测方程为z_k = d_测(k) = H + h_veh(k) - h_track(k) + v_k。

### 5.3 三源融合判级与数据上传

综合检测系统在缺陷等级判定中采用三源融合方法：2D图像提供视觉特征，3D点云提供轮廓几何数据，IMU提供姿态振动信号。三源融合输出缺陷类型、置信度和判定等级，当置信度低于设定阈值时进入人工复核流程。

检测数据在统一时空基准下存储，帧编号、里程坐标和UTC时间戳三键索引，支持精确回溯和空间查询。数据通过4G/5G网络实时上传至远程监控平台，异常告警秒级送达。
