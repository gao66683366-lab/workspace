# 第1章 项目概述与核心定位

## 1.1 项目背景

列车底部构件是轨道交通车辆运行安全的关键部位，包含轮对、轴箱、制动系统、液压系统、传动机构、车体底架等核心组件。这些部件的隐性损伤（如裂纹、松动、渗漏等）难以通过人工目视快速发现，传统人工巡检依赖检修人员钻入车底，面临挤压、磕碰、触电等高危作业风险，且检测效率低（单节车厢>=90分钟）、标准化程度差、检测结果难以追溯。

随着铁路运维向数字化、智能化转型，迫切需要一种能够替代人工钻车底高危作业、实现车底无人化智能检测的专用装备。

**列车底部构件检测机器人**正是在这一背景下提出，面向轨道交通车辆检修领域，集成高清视觉采集、铁路轨道轨底运动平台、多传感器融合、边缘计算与AI缺陷识别技术，实现车底关键构件全覆盖、无死角、数字化检测。

## 1.2 核心定位

本产品是面向轨道交通车辆检修的**特种移动式智能检测装备**，搭载铁路轨道轨底运动平台，替代人工完成车底高危检测作业。

**产品定位三要素：**

| 定位要素 | 内容 |
|---|---|
| 使用场景 | 地铁、动车组、普速客车/货车车辆段；铁路大机段工程车辆整备场 |
| 核心价值 | 无人化 · 智能化 · 数字化 |
| 目标用户 | 车辆段检修车间、工务段、大机段设备管理部门 |

## 1.3 五大核心价值

| 序号 | 核心价值 | 说明 |
|---|---|---|
| 01 | **无人化作业** | 替代人工钻车底高危作业，全流程无需人员进入车底区域 |
| 02 | **智能化检测** | AI实时缺陷识别（裂纹/锈蚀/渗漏/松动/磨损），边检边报 |
| 03 | **数字化存证** | 每帧图像带时间/距离/位置水印，数据自动归档可追溯 |
| 04 | **高效化作业** | 单车检测<=20分钟，效率提升>=96% |
| 05 | **全车型覆盖** | 一套系统覆盖地铁、动车、客车、货车、大机段工程车辆 |
