## 第一部分：姿态传感器方案不可行理由

### 1.1 问题定义

铁路轨道高低检测的目标：获取左轨、右轨各自的高程序列，计算左右轨高差，并按波长（短波 <10m / 长波 ≥10m）进行轨道质量评价。

### 1.2 姿态传感器能力边界

| 能测（✅） | 不能测（❌） |
|-----------|-------------|
| 车体俯仰角（pitch）| 左轨绝对高程 |
| 车体横滚角（roll）| 右轨绝对高程 |
| 车体垂向加速度 | 左/右轨独立高程序列 |
| 车身倾斜程度 | 车体跳动与轨道不平顺的分离 |

### 1.3 四个核心不可行原因

**原因一：陀螺仪输出角速度，不是位移**

- 陀螺仪输出 °/s（角速度），积分一次得角度（°），再积分得角位移
- 角位移 ≠ 垂向位移；陀螺仪没有能力感知空间中沿激光方向的绝对位移
- 积分漂移随时间累积，10 秒后姿态误差可达毫米至厘米级

**原因二：车身姿态 ≠ 轨道高程**

```
姿态传感器装在车体上
        ↓
输出"车身倾斜了多少度"
        ↓
无论精度多高、装几个传感器
都无法直接回答"左轨比右轨高多少"这个问题
```

举例：车身姿态平稳，但左轨在缓慢下沉，姿态传感器输出为 0，轨道病害已被漏检。

**原因三：无法区分左轨和右轨**

- 一套 IMU 装在车体中间，测的是中间那一点的车体姿态
- 1435mm 轨距，左轨正上方和右轨正上方是两个不同的物理位置
- 左轨、右轨各自的高程，在一套 IMU 里无法分离为两路独立信号

**原因四：跳动干扰无法自消除**

- 车体跳动的频率（1~5Hz）与中波轨道不平顺（波长 1~10m，速度 1m/s 对应 0.1~1Hz）频段相邻
- IMU 测到的振动里同时包含"车跳了"和"路不平"两部分，自身无法分离
- 没有独立的空间参考基准，无法将两者剥离

### 1.4 结论

> **纯姿态传感器方案仅能提供车体姿态参考信息，不能独立完成左右轨独立高程测量任务。必须引入直接测量轨廓的传感器（3D 线激光）作为主测手段，IMU 仅作为辅助校正使用。**

## 第二部分：完整技术方案（修正版）

### 2.1 传感器配置

| 序号 | 传感器 | 数量 | 角色 | 输出 |
|------|--------|------|------|------|
| 1 | 3D 线激光传感器 | 2 台 | 主测 | 左右轨轨头廓形点云（X=里程, Y=横, Z=高程） |
| 2 | 双轴倾角传感器 | 1 套 | 辅助校正 | 车体俯仰角 θ_pitch、横滚角 θ_roll |
| 3 | 高性能加速度计 | 1 套 | 辅助校正 | 垂向加速度 a_vert |
| 4 | 轮速编码器 | 1 套 | 里程/速度基准 | 实时速度 v(t) |
| 5 | 工业控制计算机 | 1 台 | 融合计算 | EKF 状态估计、TQI 统计 |

### 2.2 安装位置几何关系

```
车身纵轴（俯仰旋转轴，沿行驶方向）：

左激光 ←———————间距 L_Laser———————→ 右激光
   ↓                                          ↓
扫描左轨头                               扫描右轨头
   ↑                                          ↑
  离轨面 H_L                               离轨面 H_R
（H_L ≈ H_R ≈ 180mm，安装时尽量等高）

[倾角传感器]  ←→  安装在车体中部（车身质心附近）

注：左激光和右激光沿车身纵轴有间距 L_Laser（典型值约 300~500mm）
    俯仰校正时不能把同一个 θ_pitch 同时用于左右两个激光
```

### 2.3 物理模型与符号约定

**坐标系定义：**

| 符号 | 含义 | 单位 |
|------|------|------|
| z = 0 | 基准参考面（海平面） | — |
| h_track(t) | 轨道表面高程（相对基准） | mm |
| h_veh(t) | 车体跳动位移（向上为正） | mm |
| H | 激光器安装高度（相对车体基准点） | mm |
| d(t) | 激光传感器测量距离（发射器到轨面） | mm |

**正确的测距方程：**

```
d(t) = 激光器位置高度 - 轨道表面高程
     = (H + h_veh(t)) - h_track(t)
     = H + h_veh(t) - h_track(t)

整理：
h_track(t) = H + h_veh(t) - d(t)

⚠️ 注意符号：h_veh 和 h_track 在公式中符号相反
   不是"高程+跳动"，而是"跳动-高程"
```

---

### 2.4 跳动频率分析

| 类型 | 频率范围 | 幅值 | 与系统关系 |
|------|---------|------|-----------|
| 高频跳动 | 10~50Hz | 0.1~2mm | 陷波滤波可剔除 |
| 中频振动 | 2~10Hz | 0.5~5mm | 与车体悬挂共振频段重叠 |
| 低频起伏 | 0.1~2Hz | 1~10mm | ⚠️ 与长波不平顺频段部分重叠 |
| 车体固有振动 | 1~5Hz | 0.5~3mm | 由悬挂系统决定，需标定 |

**波长-频率换算（v = 1m/s）：**

```
λ = v / f

λ = 10m  → f = 0.10 Hz  （长/短波分界线）
λ = 1m   → f = 1.00 Hz  （中波）
λ = 0.1m → f = 10.0 Hz

结论：
1. 短波（<10m）→ f > 0.1Hz，和高频跳动有一定分离窗口
2. 中波（1~10m）→ f = 0.1~1Hz，和低频车体起伏（0.1~2Hz）有重叠
3. EKF 的状态分离能力是解决频段重叠问题的核心手段
```

### 2.5 状态空间模型（修正版）

**状态向量（4维）：**
```
X(t) = [h_track(t), v_track(t), h_veh(t), v_veh(t)]ᵀ
     = [轨道高程, 轨道高程变化率, 车体跳动位移, 车体跳动速度]ᵀ
```

**状态转移矩阵（修正：速度进入 F 矩阵）：**

```
X(k+1) = F(k) · X(k) + W(k)

     1   v·Δt   0    0
F =  0    1     0    0
     0    0     1   v·Δt
     0    0     0    1

其中：
v     — 轮速编码器实时速度（m/s），每个时刻更新
Δt    — 采样时间间隔（s）
W(k)  — 过程噪声，W(k) ~ N(0, Q)
```

⚠️ **修正说明：** F 矩阵里的 Δt 改为 v·Δt = 空间位移量，使得状态预测步能正确反映车体在空间中的实际运动关系。

**观测方程（修正符号）：**
```
Z(k) = d(k)
     = H + h_veh(k) - h_track(k) + V(k)
     = H + X₃(k) - X₁(k) + V(k)
     = C · X(k) + H + V(k)

其中：
C = [-1, 0, 1, 0]
V(k) ~ N(0, R)  — 观测噪声（R 由激光测距精度决定）
```

### 2.6 俯仰角校正（修正：左右激光分别计算）

车体俯仰时，左右激光的实际校正量不同：

```
设：
θ_pitch(k)  — k时刻倾角传感器输出的俯仰角
L_Laser      — 左右激光沿车身纵轴的间距（m）
L_sensor     — 倾角传感器到车身纵轴中心的偏移（m）

激光器 i（i=左或右）的等效俯仰校正角：
θ_i(k) = θ_pitch(k) + L_i · pitch_rate(k) / v(k)

其中：
L_i        — 激光 i 相对于俯仰旋转中心的纵向距离
pitch_rate — 俯仰角变化率（°/s），由倾角传感器角速率输出
v(k)       — 轮速编码器速度（m/s）

余弦校正公式：
d_i_校正(k) = d_i_测(k) / cos(θ_i(k))

⚠️ 当 θ_i = 0.1° 时，校正量 ≈ 0.76mm，不可忽略
⚠️ 倾角传感器通常安装在车身中部，不在激光器位置
   必须根据安装几何分别计算左右等效俯仰角
```

### 2.7 扩展 Kalman 滤波算法（EKF）

**参数初始化（新增）：**

```
初始状态：
X̂(0) = [d(0) - H, 0, 0, 0]ᵀ
      — 初始高程 = 首帧激光读数减去安装高度
      — 初始速度、跳动均设为 0

初始协方差矩阵（对角）：
P(0) = diag([σ_h², σ_v², σ_h_veh², σ_v_veh²])
     = diag([1.0², 0.1², 0.5², 0.1²])  单位：mm

过程噪声协方差矩阵 Q（由 Allan 方差标定）：
Q = diag([q_h, q_v, q_h_veh, q_v_veh])
  — q_h_veh 和 q_v_veh 反映车体跳动动态特性
  — 通过加速度计静态 Allan 方差实验确定

观测噪声协方差 R：
R = σ_laser²
  — σ_laser 为 3D 激光测距精度（±0.1mm → σ ≈ 0.05mm）
```

**EKF 预测步：**
```
X̂⁻(k) = F(k-1) · X̂(k-1)
P⁻(k)  = F(k-1) · P(k-1) · F(k-1)ᵀ + Q(k-1)
```

**EKF 更新步（融合倾角传感器和加速度计）：**

```
# 1. 观测残差（使用校正后的激光数据）
ŷ(k) = d_校正(k) - [H + X̂⁻₃(k) - X̂⁻₁(k)]

# 2. 倾角传感器辅助更新（当 |θ_pitch| 较大时启用）
如果 |θ_pitch(k)| > θ_threshold：
    # 用倾角信息修正 h_track 的梯度估计
    K_pitch = [k₁, k₂, 0, 0]ᵀ  # 仅修正前两个状态
    X̂(k) = X̂⁻(k) + K_pitch · ŷ(k)
否则：
    # 标准 Kalman 更新
    S(k) = C · P⁻(k) · Cᵀ + R
    K(k) = P⁻(k) · Cᵀ · S(k)⁻¹
    X̂(k) = X̂⁻(k) + K(k) · ŷ(k)
    P(k)  = [I - K(k) · C] · P⁻(k)

# 3. 加速度计辅助ZUPT（极低速时触发，见 2.8 节）
```

**关键输出：**
```
X̂₁(k) = h_track(k)  — 轨道高程（跳动已剥离）
X̂₂(k) = v_track(k)  — 轨道高程变化率
X̂₃(k) = h_veh(k)    — 车体跳动位移（单独输出，供车体检修参考）
```

### 2.8 ZUPT 的适用范围（修正）

```
ZUPT 触发条件分析：
v_sys = 1 m/s（系统正常运行速度）

ZUPT 要求 |v_veh(k)| < v_threshold 才触发

如果 v_threshold = 0.01 m/s：
→ 相当于车体几乎停止（< 3.6cm/s）才能触发
→ 正常运行时 v_veh 是 0~5mm 的高频振动，不是低频速度漂移
→ ZUPT 在本系统中基本无效！

ZUPT 实际适用于：静止检测台、极低速标定场景
```

**修正后的策略：**
```
本系统主要依赖 EKF 本身的状态分离能力（X̂₁ 和 X̂₃ 分别估计）
ZUPT 仅在以下特殊场景使用：
1. 标定停车时（v < 0.01 m/s），用 ZUPT 修正初始偏置
2. 意外急停时，防止 Kalman 滤波发散

加速度计的作用改为：
→ 提供高频振动噪声的统计特性（Allan 方差）
→ 用于标定 Q 矩阵的过程噪声参数
→ 不依赖 ZUPT，而是作为连续的噪声约束输入 EKF
```

### 2.9 车体高频跳动抑制

**2.9.1 自适应陷波滤波**

```
陷波滤波器（Notch Filter）—— 剔除车体固有振动频率 f_veh：

f_veh 由车体悬挂系统决定（标定得到，典型值 1~5Hz）

def adaptive_notch(signal, f_veh, fs, Q=30):
    w0 = 2π · f_veh / fs
    α = sin(w0) / (2Q)
    
    b = [1, -2cos(w0), 1] / (1 + α)
    a = [1, -2cos(w0), 1 - α]
    
    return lfilter(b, a, signal)

⚠️ 陷波滤波只剔除 f_veh 附近的一个窄频带
   不能去除更低频的跳动（0.1~1Hz）
   那部分依赖 EKF 的状态分离
```

**2.9.2 滑动窗口多项式趋势项去除**

```
目的：从 h_track 估计序列中，分离出车体缓慢起伏（趋势项）
      和真正的轨道不平顺（波动项）

窗口长度：L = 200 采样点
         @fs = 2000Hz，v = 1m/s
         → L × (v/fs) = 200 × (1/2000) = 0.1m

拟合阶数：n = 2（二次多项式）

对窗口内信号 y = [d(1), d(2), ..., d(L)]

最小二乘拟合：
y_fit(k) = a₀ + a₁·k + a₂·k²   (k = 1, 2, ..., L)

输出：
趋势项 = y_fit              — 车体缓慢起伏（弃用）
波动项 = y - y_fit          — 真正的轨道不平顺信号（保留）
```

### 2.10 3D 激光廓形高程提取

**2.10.1 廓形点云处理**

```
每帧廓形点集：P = {(x_i, y_i, z_i), i = 1...N}

步骤1：离群点去除（统计滤波）
        对每个点，计算其到 k 近邻的平均距离
        删除距离 > 2σ 的点

步骤2：光顺滤波（中值滤波，窗口 3 点）

步骤3：轨头边界检测
        y 方向梯度 |∂z/∂y| 最大的两个点
        → 左边界 y_L，右边界 y_R

步骤4：亚像素轨头最高点定位（二次曲线拟合）

        在 z 方向最大值点附近取 5 个点
        拟合：z(y) = a·y² + b·y + c
        峰值：y_peak = -b/(2a)
             z_peak = z(y_peak)

        精度：从 ±1 像素提升到 ±0.1 像素
```

**2.10.2 亚像素精度提取公式**

```
设峰值附近三点：(y₁,z₁), (y₂,z₂), (y₃,z₃)，y₂ 为整数像素位置

z(y) = a·y² + b·y + c

由三点坐标解出 a, b, c：
y_peak = -b/(2a)              → 亚像素峰值位置
z_peak = z(y_peak)            → 亚像素峰值高程

典型精度：
像素分辨率：0.05mm/pixel
亚像素精度：±0.005mm（约 ±0.1 pixel）
```

### 2.12 Allan 方差参数标定流程

```
目的：通过加速度计和陀螺仪的静态测试，确定 EKF 的 Q、R 噪声参数

步骤：
1. 静止采集 N 分钟（>10min）加速度计原始数据 a_vert(t)
2. 计算 Allan 方差曲线 AV(τ)
3. 从 AV(τ) 曲线特征区识别噪声源：
   τ = 1s   附近 → 角度随机游走（√Q 的来源）
   τ = 10s  附近 → 零偏不稳定性（主要噪声）
   τ > 100s → 趋势项（温漂等）
4. 将识别结果映射到 Q、R 矩阵对应元素

典型值（参考 HWT9073 + 高性能加速度计组合）：
q_h_veh ≈ 0.01 mm²/s  （车体跳动过程噪声）
q_v_veh ≈ 0.001 mm²/s² （车体跳动速度过程噪声）
R ≈ (0.05mm)² = 0.0025 mm²  （激光测距噪声）
```
