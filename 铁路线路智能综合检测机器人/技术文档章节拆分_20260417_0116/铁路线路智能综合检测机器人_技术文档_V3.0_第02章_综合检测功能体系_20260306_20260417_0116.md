## 02 综合检测功能体系
### 一、八大核心检测功能

一台车一次通过，全部 8 项检测数据同步输出，数据同源、同时、同标准。

| 序号 | 检测功能 | 搭载传感器 | 检测原理 | 输出结果 |
|---|---|---|---|---|
| 01 | 轨距检测 | 多传感器融合 | 综合左右轨 3D 点云数据计算实际轨距值，与标准轨距（1435 mm）比对得出偏差值 | 轨距实际值（mm）/ 偏差值（mm）/ 超限标记 |
| 02 | 水平检测 | 姿态传感器HWT905 | 姿态传感器HWT905实时测量车体水平姿态角，结合里程信息计算左右轨高差，判定水平偏差 | 水平偏差值（mm）/ 左右倾斜角（°）/ 平顺性指标 |
| 03 | 高低检测 | 单点测距传感器 + 姿态传感器HWT905 | 陀螺仪数据积分计算车体俯仰角变化，结合里程信息计算高低不平顺值 | 高低不平顺值（mm）/ 长波偏差 / 短波偏差 |
| 04 | 轨面缺陷检测 | 2D 工业相机 ×2 | 工业相机垂直拍摄轨面，AI 图像语义分割识别裂纹/掉块/凹陷，输出缺陷位置（里程坐标）和损伤等级 | 裂纹位置 + 掉块位置 + 凹陷区域 + 损伤等级（1~3 级） |
| 05 | 道钉/螺栓检测 | 2D 工业相机 ×2 | 相机拍摄轨枕区域，AI 目标检测算法识别道钉/螺栓状态，统计缺失/松动/歪斜数量 | 缺失数量 + 松动数量 + 歪斜数量 + 里程分布统计 |
| 06 | 3D 钢轨轮廓检测 | 3D 线激光 ×2 | 线激光垂直向下扫描轨头，获取轨头三维点云数据，与标准廓形比对计算磨耗量和横移量 | 磨耗量（mm）/ 横移量（mm）/ 廓形偏差（mm）/ 超限标记 |
| 07 | 钢轨波磨检测 | 3D 线激光 + 相机 | 线激光扫描获取轨面高程数据，进行频谱分析提取波长和波深，结合里程信息标记波磨区段位置 | 波长（mm）/ 波深（mm）/ 分布里程 / 劣化程度（轻/中/重） |
| 08 | 钢轨焊缝检测 | 2D 工业相机 ×2 | 相机对钢轨接头区域成像，AI 分类识别焊缝缺陷类型，定位缺陷位置 | 缺陷类型（光斑/气孔/夹渣/未熔合等）/ 等级 / 里程位置 |

### 二、为什么我们是"综合检测"

**单一功能检测的时代已经过去。**

传统铁路检测依赖人工巡检或单功能检测设备，每次只能完成一项检测任务。以钢轨巡检为例，探伤仪检测钢轨内部伤损，轨检仪检测轨距与水平，晃车仪检测车体振动响应——每套设备专职一项，需要多次出车、多次上道、多人协同，数据相互孤立，人员协调难度极大。更重要的是，单一功能的检测只能发现局部问题，无法从多维度交叉验证，同漏同误的风险高。

本系统的核心定位是**综合检测**，不是把多个传感器简单装在同一台车上拼凑功能，而是从系统层面构建了一套完整的多传感器协同、多算法融合、多维度判定的综合检测体系。本系统的目标不是"能用"，而是"好用"——让检测数据真正产生综合价值。

综合检测带来三大核心价值：

**（1）数据同源**

同一台设备、同一次出行、同一个里程坐标，同步完成 8 项检测。相机图像和激光点云天然对齐，姿态参数与缺陷位置一一绑定，不同传感器数据之间可以交叉印证，从根本上消除了多套设备数据无法对齐的难题。当发现一处疑似裂纹时，可以同时查看该位置的3D廓形是否变形、姿态角是否异常，三维印证，大幅提升判级准确性。

**（2）一次完成**

一台车跑完全部 8 项检测，无需多次上道、无需多人配合，以单次 2 小时作业为例，人工巡检同等时间仅能覆盖约 3 km，且数据质量受人员状态影响；本系统覆盖约 6 km，8 项数据一次性获取。作业组织复杂度从"多人多天"降为"单人一次"。

**（3）综合判级更可靠**

单一维度的检测结果容易被干扰。如仅靠 2D 图像可能将阴影或污渍误判为裂纹；仅靠 3D 点云可能遗漏轻微表面裂纹；仅靠姿态参数无法定位具体病害位置。本系统综合图像（2D）+ 轮廓（3D）+ 姿态（姿态传感器HWT905）三个维度联合判定，任何单一维度的异常都需要其他维度印证才能确认，大幅降低误报率和漏报率。

### 三、综合检测 vs 单一功能检测

| 对比维度 | 传统单功能设备 | 本系统（综合检测） |
|---|---|---|
| 检测覆盖 | 单次仅支持 1 项 | **一次出动，8 项全覆盖** |
| 作业组织 | 多次出动、多人分工协作 | **一次上道，单人操作完成全部** |
| 数据关联性 | 各模块数据孤立，无法融合 | **同源数据统一时空基准，交叉印证** |
| 判级可靠性 | 单维度判定，误报率和漏报率高 | **三源融合判级，综合准确率大幅提升** |
| 全生命周期成本 | 每套设备独立采购，维护成本高 | **一机多用，综合成本最优** |
| 夜间作业能力 | 差，依赖光照条件 | **强，全天候连续作业，不受光线影响** |
| 部署复杂度 | 多套设备多次部署，协调难度大 | **一台设备一次部署，即刻运行** |
| 数据追溯性 | 各设备数据格式不统一，难以追溯 | **统一 Frame ID + 里程 + 时间戳，全程可追溯** |

### 四、多维数据融合智能判级

单一维度的检测结果容易被环境噪声或偶发因素干扰，综合多个维度的数据才能得出准确结论。

系统采用图像（2D）+ 轮廓（3D）+ 姿态（姿态传感器HWT905）三源数据联合判定，以融合节点为中枢，对每处病害进行综合评分：

| 检测案例 | 2D 图像结果 | 3D 廓形结果 | 姿态参数结果 | 综合最终结论 |
|---|---|---|---|---|
| 案例 A | 疑似裂纹 | 无轮廓变形 | 姿态无异常 | **降级：疑似，建议人工复核**（仅图像异常，单维度不可靠，降级处理） |
| 案例 B | 明显裂纹 | 局部凹陷变形 | 同位置水平、高低异常 | **升级：严重缺陷，重点处置**（三源同时报警，置信度极高，立即升级处理） |
| 案例 C | 图像正常 | 磨耗量超标 | 姿态无异常 | **直接输出二级报警**（廓形数据独立超限，无需图像确认） |

**综合判级核心优势**：案例 B 中，如果仅看图像可能被判定为普通裂纹，但综合 3D 廓形发现局部凹陷变形，且同位置姿态参数也出现异常，三源互相印证，直接升级为严重缺陷。这种跨维度验证大幅提升了判级的准确性，避免了单一维度的误判和漏判。

### 五、病害四级判定体系

系统建立四级病害判定标准，与铁路工务标准直接对接：

| 等级 | 含义 | AI 判级说明 | 人工处置建议 |
|---|---|---|---|
| **正常** | 各项指标在允许限值范围内，线路状态良好 | AI 三源数据综合评分在安全阈值内 | 正常巡检，维持现状 |
| **一级** | 轻微异常，接近阈值 | 至少一个维度触发一级预警，三源综合评分处于注意区间 | 持续观察，纳入重点跟踪计划 |
| **二级** | 明显超限，超出规范允许范围 | 至少一个维度明确超限，三源综合评分触发告警 | 纳入计划养护，优先安排处置 |
| **三级** | 严重超限，已危及行车安全 | 多维度同时严重超限，或单一维度极端超限 | 须立即处置，启动应急养护流程 |
