# 第11章 验收与交付标准

## 11.1 验收流程总览

验收分为五个阶段：外观验收 → 功能演示 → 性能测试 → 效果验证 → 综合验收。

## 11.2 外观验收

| 验收项目 | 验收内容 | 判定标准 |
|---|---|---|
| 设备外观 | 表面无划伤、无锈蚀、无变形 | 目视检查 |
| 铭牌标识 | 设备编号、型号、厂家标识 | 标识清晰可读 |
| 配件清单 | 主机、手持终端、线缆、备件 | 清单一致 |
| 随机文档 | 操作手册、维护手册、合格证 | 文档齐全 |

## 11.3 功能演示验收

| 功能项 | 验收内容 | 判定标准 |
|---|---|---|
| 轨底平台行走 | 空载状态下前进/后退/转向 | 运动平稳，无卡滞 |
| 云台控制 | 水平360°、垂直±180° | 无死角，控制响应正常 |
| 相机成像 | 开机采图，检查画面清晰度 | 4K画面清晰，低照度下可辨 |
| 照明系统 | 亮度可调，频闪同步正常 | 无频闪条纹 |
| 避障功能 | 模拟障碍物测试急停 | <=50ms内制动 |

## 11.4 性能测试验收

| 指标项 | 测试方法 | 目标值 | 判定 |
|---|---|---|---|
| 行走速度 | 计时测速，0–0.5m/s分档测试 | 0–0.5 m/s | 达标/不达标 |
| 定位精度 | 在轨道上标记位置，行走后读数 | ±20 mm | 达标/不达标 |
| 云台精度 | 用量角器测量云台转动角度误差 | <0.1° | 达标/不达标 |
| 越障能力 | 放置30mm障碍物，机器人通过测试 | <=30mm通过 | 通过/不通过 |
| 爬坡能力 | 30°坡道行走测试 | <=30°通过 | 通过/不通过 |
| 续航时间 | 满电连续作业计时 | >=4h | 达标/不达标 |
| 防护等级 | IP67验证（浸水30min） | IP67 | 通过/不通过 |
| 工作温度 | 高低温箱测试 | -20°C~+60°C正常 | 达标/不达标 |
| 视频延迟 | 示波器测量相机曝光到显示延迟 | <200ms | 达标/不达标 |
| AI识别响应 | 示波器测量推理延迟 | <200ms | 达标/不达标 |

## 11.5 效果验证

| 验证项 | 验证内容 | 判定标准 |
|---|---|---|
| 采图成像质量 | 实车车底采集图像 | 4K清晰，无严重噪声/反光 |
| AI识别效果 | 在测试图像集上验证 | 准确率>=95% |
| 报告生成完整性 | 自动生成报告，核查内容 | 内容完整，格式正确 |
| 系统对接 | 对接MRO/EAM测试 | 接口正常，数据完整 |
