# 第5章 人机交互与操控系统

## 5.1 设计理念

人机交互系统的设计以**安全第一、操作简便、耐用可靠**为核心理念：

- **安全优先：** 所有操控设计均以保障人员和设备安全为最高优先级
- **一体化触控：** 工业级触控屏同一界面集成所有操控，操作直观
- **大尺寸防误触：** 面向工业现场戴手套操作场景，所有控件大尺寸、图标化
- **零学习成本：** 界面设计遵循铁路工控标准，操作逻辑与现有检修习惯一致

## 5.2 操控终端规格

本系统采用**嵌入式工业级触控显示屏（HMI）**作为主操控终端。

| 项目 | 参数 |
|---|---|
| 屏幕尺寸 | 10–15寸 |
| 分辨率 | 1920×1080 或更高 |
| 触控类型 | 工业电容屏，多点触控，戴手套可操作 |
| 防护等级 | IP65 |
| 安装方式 | 壁挂式 / 操作杆手持式 |
| 通信 | WiFi 6（与主控实时通信） |
| 工作温度 | -20°C ~ +60°C |

## 5.4 各功能区说明

### 5.4.1 主显示区

| 功能 | 说明 |
|---|---|
| 实时视频 | 4K@30fps实时画面，占界面>=60% |
| 数字放大 | 单指滑动局部放大，最高x32 |
| 画中画 | 主画面+小画面双路同显 |
| 画面冻结 | 一键冻结定格，便于细看 |

### 5.4.2 小车行走控制

| 控件 | 功能 |
|---|---|
| 前进按钮 | 按住不放前进，松开停止 |
| 后退按钮 | 按住不退，松开停止 |
| 停止按钮 | 立即停止 |
| 速度滑条 | 0–0.5m/s无级调速 |

### 5.4.3 云台控制

| 控件 | 功能 |
|---|---|
| 云台左/右 | 控制前相机水平方向 |
| 云台俯仰 | 控制前相机垂直方向 |
| 预设位 | 一键切换预设角度 |

### 5.4.4 光源控制

| 控件 | 功能 |
|---|---|
| 亮度滑条 | 0–100%无级调节 |
| 频闪同步 | 照明与相机曝光同步 |

### 5.4.5 采图控制

| 控件 | 功能 |
|---|---|
| 一键采图 | 单次抓拍高清原图，自动水印归档 |
| 连续采图 | 按距离/时间间隔自动批量采图 |
| 图像预览 | 当前帧缩略图显示，可查看/删除 |

### 5.4.6 状态栏

| 显示项 | 内容 |
|---|---|
| 设备状态 | 运行/停止/故障 |
| 通信状态 | WiFi信号强度 |
| 电量 | 剩余电量/续航时间 |
| 里程 | 已行走距离 |
| 当前模式 | 枕木模式/轨底模式 |

## 5.5 界面设计要求

| 需求项 | 说明 |
|---|---|
| 布局原则 | 视频画面为主（>=60%），操控面板为辅 |
| 操控优化 | 大按钮（>=44pt），戴手套可操作 |
| 信息不重叠 | 关键数据一眼可见，不相互遮挡 |
| 风格标准 | 铁路工控风格，专业简洁 |
| 响应速度 | 界面操作响应<=100ms，无卡顿 |
| 界面稳定 | 长时间运行不闪退、不花屏 |

## 5.6 物理急停

物理急停按钮为独立硬件，不依赖触屏操作，在任何情况下均可触发：

| 控件 | 优先级 | 功能 |
|---|---|---|
| 急停按钮（红色） | 全局最高优先级 | 立即切断电机驱动，机器人停止 |
