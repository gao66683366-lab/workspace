## 今日核心工作：传感器方案口径对齐

### 问题识别
MEMORY.md与实际系统存在重大不一致：
- MEMORY.md记录"高低检测：3D线激光（主测）+HWT905（辅助）"
- 实际已确认方案：单点测距传感器（主测）+HWT905（辅助），没有三角测量

### 确认后的正确方案
| 检测功能 | 传感器 | 原理 |
|---|---|---|
| 轨面缺陷（01）| 2D工业相机×2 | 机器视觉 |
| 道钉/螺栓（02）| 2D工业相机×2 | 机器视觉 |
| 钢轨轮廓（03）| 3D线激光×2 | 激光三角测量 |
| 波磨检测（04）| 3D线激光×2 | 激光三角测量 |
| 焊缝缺陷（05）| 2D工业相机×2 | 机器视觉 |
| 轨距检测（06）| 测距传感器矩阵 | 直接测量，与姿态无关 |
| 水平检测（07）| HWT905横滚角 | 直接输出，不是单点测距 |
| **高低检测（08）** | **单点测距×2 + HWT905** | **直接测距减法：z = H - D** |

### 高低检测核心公式（已修正）
- 测量公式：$z(s) = H - D(s)$（直接测距减法，无三角测量）
- 俯仰角校正：$D_{corrected}(s) = D_{raw}(s) \cdot \cos\theta_p - (H - D_{raw}(s)) \cdot \sin\theta_p$（方向正确版）
- EKF状态向量：$[h_{track}, h_{veh}, v_{veh}]$（3状态，已移除冗余v_track）

1. "单点测距传感器"是唯一的高低检测主测手段，不能与3D激光混淆
2. 水平检测完全由HWT905横滚角输出，与单点测距无关
3. 轨距检测由测距传感器矩阵直接计算，与姿态传感器无关
4. 论文标题去掉"3D激光"，改为"混合传感器融合"
5. EKF状态向量从5状态$[θ_p, θ_r, z_L, z_R, v]$改为3状态$[h_{track}, h_{veh}, v_{veh}]$
