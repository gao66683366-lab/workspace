## 10 设备完好性保障与远程监控

### 一、设备完好性保障体系

在检测过程中，对机器人上所有检测设备进行实时完好性监控，确保设备始终处于良好工作状态。设备安全稳定运行是检测数据质量的根基。

```
┌─────────────────────────────────────────────────┐
│           检测设备完好性实时监控                  │
│                                                  │
│  6路工业相机 ──→ 在线状态 / 帧率 / 丢包率 / 温度 │
│  2路3D激光 ──→ 功率 / 数据率 / 温度             │
│  2路陀螺仪 ──→ 角速度 / 姿态角 / 健康状态        │
│  工控机本体 ──→ CPU / 内存 / 磁盘 / 进程 / 网络  │
│  工业总线 ────→ 通信状态 / 故障码               │
│  通信链路 ────→ 移动网络信号 / 带宽 / 延迟       │
└─────────────────────────────────────────────────┘
```

**完好性监控指标：**

| 设备 | 监控参数 | 告警阈值 |
|---|---|---|
| 工业相机 ×6 | 帧率 / 丢包率 / 在线状态 / 温度 | 帧率 <20fps / 丢包率>1% / 离线 |
| 3D 激光 ×2 | 数据率 / 温度 / 功率 | 数据率异常 / 温度超限 |
| 双陀螺仪 | 角速度漂移 / 姿态角异常 | 漂移超限 |
| 工控机 | CPU占用>80% / 内存>85% / 磁盘<10GB | 超限告警 |
| 工业总线 | 通信状态 / 故障码 | 通信中断 / 故障码非零 |
| 移动网络链路 | 信号强度 / 带宽 / 延迟 | 信号<-100dBm / 延迟>500ms |

### 二、智能安全供电管理

用电安全管理是保障机器人安全运行的核心，贯穿检测任务全流程。

本系统的智能安全供电管理体系覆盖所有用电节点，实现从蓄电池到各传感器设备的全链路电流、电压、温度实时监控，确保设备在检测过程中不会因电力问题而受损或停机。

**蓄电池安全管理 — 全参数实时监控**

蓄电池是检测机器人的"心脏"，本系统对蓄电池实施全参数实时监控：

| 监控参数 | 说明 | 安全机制 |
|---|---|---|
| **电压** | 实时监控蓄电池组电压 | 低于阈值立即报警并触发返航，防止过放 |
| **电流** | 充放电电流实时监测 | 充电过流自动断充，放电异常预警 |
| **SOC 电量** | 剩余电量百分比实时显示 | 电量低于预设阈值前主动报警 |
| **SOH 健康度** | 电池健康状态评估 | 健康度低于阈值时提示更换 |
| **续航预估** | 基于当前电量 + 历史能耗 | 实时预估剩余可行驶里程，提前规划返航 |
| **BMS 电路板温度** | 蓄电池管理系统电路板温度实时采集 | 电路板温度超限立即报警，防止BMS失效引发安全事故 |
| **智能电路管理板温度** | 智能电源管理板温度实时采集 | 管理板温度超限自动降载或断电，防止过热损坏 |

**三级保护机制：**

| 级别 | 触发条件 | 动作 |
|---|---|---|
| 预警 | 电量 <20% 或 电压接近低限 | 云端推送预警，提醒准备返航 |
| 告警 | 电量 <10% 或 温度超限 | 声光报警 + 强制触发一键返航 |
| 保护 | 电压/电流严重超限 | 硬件独立保护回路立即切断供电，保护设备和线路 |

> BMS 电路板和智能电路管理板的温度实时采集，是防止蓄电池管理系统自身失效引发安全事故的最后一道防线，填补了传统监控方案的关键盲区。

**检测设备用电管理 — 全节点电流电压实时监控**

整机所有用电节点均纳入实时监控，任何电流电压异常均可快速定位：

| 用电节点 | 监控参数 | 说明 |
|---|---|---|
| 6路工业相机 | 电压 / 电流 / 功率 | 相机节点用电状态实时监控 |
| 2路3D激光 | 电压 / 电流 / 功率 | 激光器节点用电状态实时监控 |
| 2路双陀螺仪 | 电压 / 电流 | 陀螺仪节点用电状态实时监控 |
| 工控机 | 电压输入 / 电流 / 功率 | 主控节点用电实时监控 |
| 工业总线伺服驱动 | 电压 / 电流 / 故障码 | 运动控制节点用电和安全监控 |

> **实时反馈机制**：所有设备的电流、电压数据**每秒刷新**，异常节点可在云端**秒级定位**，大幅缩短故障排查时间。

**运动控制安全保障 — 工业总线硬实时守护**

| 监控项 | 说明 |
|---|---|
| 工业总线通信状态 | 实时监控通信是否中断 |
| 伺服驱动状态 | 实时获取驱动故障码 |
| 急停回路状态 | 硬件独立急停回路，失效导向安全 |
| 运行速度一致性 | 定速巡航保障采集数据速度稳定 |

### 三、远程监测控制能力

| 能力 | 说明 |
|---|---|
| 实时状态查看 | 云端实时查看任一设备节点的工作状态和用电数据 |
| 参数远程调优 | 远程调整相机曝光/帧率/检测阈值，无需到场 |
| 设备远程重启 | 单节点故障可远程重启恢复，减少现场出勤 |
| 故障隔离 | 单节点故障自动隔离，不影响整体任务执行 |
| 远程固件升级 | 传感器固件可通过移动网络远程推送升级 |

### 四、四类用户价值

| 用户角色 | 使用场景 | 核心价值 |
|---|---|---|
| 现场操作人员 | 出发前检查设备状态，实时监控任务进度 | 快速就绪，避免无效作业 |
| 远程监控中心 | 多台设备集中监控，大屏可视化 | 全线设备状态和用电数据一目了然 |
| 管理决策者 | 设备利用率 / 故障率 / 出勤率统计 | 排班和资源调配有据可依 |
| 运维保障团队 | 故障提前预警，远程诊断，用电安全 | 减少现场出勤，提升维护效率 |
