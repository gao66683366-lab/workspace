# 面向铁路线路的智能视觉与多模态感知融合综合检测系统研究与实现

## 第2章 系统总体设计

### 2.1 研究问题分析

由第1章分析可知，铁路线路综合检测领域存在四大共性问题：时空基准不统一、融合架构缺乏系统设计、自适应能力不足、缺乏实际线路系统性验证。这四个问题不是某一算法的局部不足，而是系统架构层面的共性瓶颈。

现有系统在架构层面分为两类：多台单功能设备分别检测、数据后期拼接——该方案各设备使用独立时钟和里程基准，融合精度受限于时钟偏差；多传感器集成方案，但融合停留在后处理层——各传感器数据流在各自时钟域独立采样，仅在结果层做简单投票或拼接，无法实现真正的特征级融合。

这两类架构的共同缺陷是：**在传感器层就没有为融合判定预留统一的时空基准**。

### 2.2 系统设计目标

针对上述问题，本文提出"综合检测系统"设计理念，核心目标有三个：

**目标一：统一时空基准。** 所有传感器在同一时钟域（PTP时间同步，200Hz全局时钟）下运行，每帧数据附带帧编号、里程坐标和UTC时间戳，从架构层面解决多源数据的时间对齐和空间关联问题。

**目标二：单设备多功能协同。** 单台检测设备集成8项检测功能，单次出行覆盖全线，作业组织复杂度从"多人多天"降为"单人一次出动"，按人·时计作业效率提升约29倍。

**目标三：数据级深度融合。** 各传感器数据在统一时空基准下实现特征级融合，而非仅在结果层拼接，通过动态加权融合算法实现跨维度的联合判定。

### 2.3 系统功能架构

系统定义8项检测功能：

| 功能群 | 序号 | 检测功能 | 主传感器 |
|---|---|---|---|
| 视觉检测群 | 01 | 轨面缺陷检测 | 2D工业相机×2 |
| | 02 | 道钉/螺栓状态检测 | 2D工业相机×2 |
| | 03 | 3D钢轨轮廓检测 | 3D线激光×2 |
| | 04 | 钢轨波磨检测 | 3D线激光×2 |
| | 05 | 钢轨焊缝检测 | 2D工业相机×2 |

### 2.4 系统架构设计

系统采用"三层两域"架构：

**传感层（感知域）：** 2D工业相机采集视觉图像（2448×2048px，20fps），3D线激光采集轮廓点云（20000Hz，3200点/轮廓），HWT905输出三轴姿态角（200Hz），单点测距传感器采集高程距离（2000Hz），测距传感器矩阵采集轨距数据（约1000Hz）。各传感器共享同一PTP时钟域。

**计算层（感知域）：** 边缘工控机（Intel Core i7，IP54防护）部署视觉AI推理引擎（YOLOv8m）和点云处理算法，实时输出缺陷检测结果、廓形参数和波磨参数。

**决策层（通信域）：** 融合服务器部署自适应动态加权融合算法，基于三键索引时空对齐后的多维检测结果，完成综合健康度评分，并通过4G/5G网络实时上传至线路健康检测平台。

### 2.5 三网物理隔离设计

系统内通信采用三网物理隔离架构，将功能性质完全不同的三类数据流部署在独立网络域：

**EtherCAT硬实时总线：** 伺服驱动控制，毫秒级响应，保障运动层的实时性要求。

**千兆以太网专网：** 相机/激光高速数据采集（视觉图像和点云数据），百兆级带宽，保障传感器层数据无损传输。

**4G/5G无线传输网：** 检测结果和健康评分数据远程上传，以及远程监控指令下发。

三网物理隔离确保了控制指令、采集数据、传输数据互不干扰，杜绝了因网络拥塞导致的控制延迟或数据丢包问题。

### 2.6 技术指标

系统核心指标：检测速度0.5m/s（精细检测）/1m/s（常规检测），每天运行3小时；轨面缺陷检测mAP@0.5达92.5%；廓形检测精度优于±0.15mm；波磨检测精度优于±0.1mm；轨距检测精度优于±0.3mm；高低检测精度优于±0.5mm；综合融合判定准确率达97.5%。
