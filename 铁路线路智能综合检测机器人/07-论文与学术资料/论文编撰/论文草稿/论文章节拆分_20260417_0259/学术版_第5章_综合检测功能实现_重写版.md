# 面向铁路线路的智能视觉与多模态感知融合综合检测系统研究与实现

## 第5章 综合检测功能实现

### 5.1 视觉检测功能群

视觉检测功能群（功能01~05）基于2D工业相机和3D线激光实现，采用"AI推理+传统图像处理"双模块并行架构。

**功能01：轨面缺陷检测。** 采用YOLOv10m目标检测模型，骨干网络Darknet配合多尺度特征金字塔（FPN），在特征图上直接预测目标类别和边界框。模型输出类别置信度和定位坐标，损失函数为分类损失与回归损失的加权和：L = L_cls + λ·L_reg。在包含裂纹、掉块、凹陷等类别的验证集上，mAP@0.5达92.5%，优于传统Faster R-CNN方案（87.3%）。

**功能02：道钉/螺栓状态检测。** 采用ZNCC（归一化相关系数）模板匹配方法。模板库预先采集标准道钉/螺栓图像作为基准，实时图像与模板做ZNCC计算：ZNCC越接近1表示匹配度越高。当ZNCC低于设定阈值时，判定为缺失或松动，缺失和松动检出率达95%以上。

**功能03：3D钢轨轮廓检测。** 3D线激光垂直朝下照射轨面，三角测量原理计算距离：D = (f·L)/x（f为焦距，L为基线距，x为光斑偏移量）。扫描频率20000Hz，每轮廓3200点。去噪采用统计离群点移除（SOR）方法，坐标变换将传感器坐标系转换至轨道坐标系（利用HWT905俯仰角和横滚角实时标定旋转矩阵），最小二乘法拟合廓形曲线。磨耗量 Δw = w_std - w_meas，精度优于±0.15mm。

**功能04：钢轨波磨检测。** 基于3D线激光采集的轨面高程序列h(n)，经FFT变换至频域：H(k) = Σh(n)·e^{-j2πkn/N}。波磨主频率f_w由频谱峰值检测确定，波长λ_w = v_speed/f_w，波深A_w = |H(k_w)|/N。轻度波磨A_w∈[0.1,0.3]mm，中度A_w∈[0.3,0.5]mm，重度A_w>0.5mm。

**功能05：钢轨焊缝检测。** 灰度阈值分割判定焊缝区域：灰度值大于焊缝阈值T_s的像素标记为焊缝区域，结合区域生长算法提取完整焊缝轮廓，判定缺陷类型并给出等级，准确率达85%以上。

### 5.2 几何参数检测功能群

几何参数检测功能群（功能06~08）基于测距传感器矩阵和HWT905实现。

**功能06：轨距检测。** 测距传感器矩阵直接测量左右轨内侧距离：g_meas = d_L + d_R（d_L为左轨内侧距离，d_R为右轨内侧距离）。该方法与车体姿态无关，不受横滚角干扰。轨距偏差Δg = g_meas - 1435mm，|Δg|>τ_g时标记超限告警，精度优于±0.3mm。

**功能07：水平检测。** HWT905横滚角直接计算超高差：Δh = 1435 × sin(θ_r)。纯姿态角计算，无需单点测距传感器参与，与高低检测的EKF融合链路完全独立，构成独立的几何参数检测通道。

**功能08：高低检测。** 单点测距传感器（2000Hz）主测，HWT905（200Hz）姿态校正辅助，采用双速EKF融合架构（详见第4章），快回路200Hz抑制车体振动，慢回路10Hz输出轨道高程估计，精度优于±0.5mm。

### 5.3 三源融合判级与四级体系

系统将8项检测结果按数据类型分为三个来源：视觉缺陷源（功能01、02、05）、激光廓形源（功能03、04）、几何参数源（功能06、07、08）。三源数据在统一时空基准下完成时空对齐后，进入自适应动态加权融合阶段（详见第4章），输出综合缺陷等级。

综合缺陷等级分为四级：**一级（正常）**——各维度检测值均在安全阈值以内；**二级（预警）**——至少一个维度超出安全阈值但未达紧急程度；**三级（警告）**——多维度联合超限，需尽快安排养护；**四级（告警）**——存在危及行车安全的严重病害，需立即处理。
