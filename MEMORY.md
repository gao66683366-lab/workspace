### 铁路线路智能检测机器人(基本参数)
- 机器人采用 4 个轮子在铁轨上运行。
- 速度(已确认):0.5 m/s（精细检测）/ 1 m/s（常规检测）,两种模式同时工作自动切换
- 每天运行 3 小时
- 0.5m/s段检测里程:约 5.4 km/天（钢轨轮廓/高低/水平/焊缝/波磨）
- 1m/s段检测里程:约 10.8 km/天（轨距/轨向/道钉/螺栓/轨面缺陷等）
- 道钉/螺栓节距:700 mm。
- 所有 2D 工业相机、3D 线激光均按垂直朝下处理。
- 两个 3D 线激光检测对象:左右钢轨轨头轮廓。
- 姿态参数传感器检测:水平(横滚角)、高低(由单点测距传感器主测,HWT905辅助校正)。
- 6 个 2D 相机安装在铁路小车车架上。
- 拍轨面的 2D 相机离钢轨约 180 mm。
- 拍道钉/螺栓的 2D 相机离轨面约 270 mm。
- 钢轨轨面宽度工程口径(已更新):80 mm。(替换旧口径 72 mm)

### 通信架构(三网物理隔离)
- EtherCAT 硬实时总线:伺服驱动控制,毫秒级响应。
- 千兆以太网:专用于相机/激光高速数据采集。
- 4G/5G 无线传输网:数据远程上传、远程监控。
- 三网物理隔离:控制网、采集网、传输网独立运行,互不干扰。

### 供电与防护
- 主供电系统:48V 大容量蓄电池组,智能电源管理。
- 工控机防护等级:IP54。
- 相机/激光防护等级:IP67。

### 工业相机规格
- 分辨率:2448×2048。
- 帧率:20~25 fps。
- 通信协议:GigE Vision。

### 姿态传感器HWT905规格(已确认)
- 采样频率:200Hz(全局时钟基准)
- 作用:姿态角输出(俯仰角/横滚角/偏航角),用于姿态校正和水平检测
- 与单点测距传感器同步:姿态传感器作为EKF融合的辅助传感器,200Hz快回路实时校正

### 单点测距传感器规格(已确认)
- 采样频率:2000Hz
- 测量原理:红外相位式/ToF,直接测距(公式:z=H-D)
- 精度:±0.05mm(实验室),±0.15mm(现场综合)
- 作用:仅用于高低检测(功能08),不参与廓形/波磨/水平/轨距等其他功能

## 今日重要结论汇总(20260421)

### 传感器职能最终配置(已确认,不得再改)
| 功能 | 主传感器 | 说明 |
|---|---|---|
| 高低检测(08) | 单点测距传感器×2 + HWT905(EKF融合) | 主测手段,直接测距公式z=H-D |
| 水平检测(07) | HWT905横滚角 | Δh=1435×sin(θ_r),与单点测距无关 |
| 廓形/波磨(03/04) | 3D线激光×2 | 独立链路,不参与EKF融合 |
| 轨距检测(06) | 测距传感器矩阵 | 直接测量,与姿态无关 |

### EKF算法(已确认)
- 状态向量: $[h_{track}, h_{veh}, v_{veh}]$ (3状态,移除冗余v_track)
- 观测方程: $z_k = d_{测}(k) = H + h_{veh}(k) - h_{track}(k) + v_k$
- 双速架构:快回路200Hz(HWT905姿态校正)/慢回路10Hz(轨道高程输出)
- 已知局限:1m≤λ<10m波段频谱重叠,EKF协方差偏大

### 关键公式修正(今日确认)
- 俯仰角校正方向: $(H-D)\cdot\cos\theta_p$ (方向已修正,非旧版D/\cos)
- 水平检测: $Δh = 1435\times\sin\theta_r$ (纯HWT905横滚角,无需单点测距)
