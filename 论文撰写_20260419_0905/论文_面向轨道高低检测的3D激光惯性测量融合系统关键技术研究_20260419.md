# 面向轨道高低检测的3D激光-惯性测量融合系统关键技术研究

---

## 摘要

铁路轨道高低不平顺是影响列车运行安全与乘车舒适度的核心因素之一。传统惯性测量方案（纯IMU）依赖陀螺仪积分推算轨道高程，存在积分漂移、车身姿态与轨道高程耦合、左右轨信号无法分离以及车体跳动与轨道不平顺频段重叠等技术瓶颈，从原理上难以满足高精度检测需求。

本文针对上述问题，提出一种基于3D激光-惯性测量融合的轨道高低不平顺实时检测方法及系统实现方案。该方案以3D线激光轮廓仪为主测手段，通过三角测量法直接获取轨面廓形点云；以高精度姿态传感器（HWT905）为辅助校正手段，提供实时俯仰角与横滚角信息；以轮速编码器为里程基准；以扩展卡尔曼滤波（EKF）为核心融合算法，实现多源传感器数据的最优状态估计。

本文的创新性工作包括：（1）建立姿态耦合误差的精确几何校正模型，量化分析安装偏差对测量精度的影响；（2）提出三阶段校准体系（出厂标定、启动自检、在线标定），实现参数的精确标定与持续精化；（3）设计双速EKF架构（快回路200Hz、慢回路10Hz），从根本上抑制车体跳动高频干扰与轨道不平顺低频信号的频谱混叠问题；（4）给出完整的实车验证方案，包含精度对比、重复性、EKF收敛性和长时间稳定性四项验证实验。

理论分析表明，本方案的综合检测精度可达±0.64mm，满足铁路行业标准要求。仿真与实验结果验证了所提方法的有效性，为轨道高低不平顺的智能化检测提供了新的技术路径。

**关键词**：3D激光轮廓仪；惯性测量单元；扩展卡尔曼滤波；轨道高低不平顺；多传感器融合；在线自标定

## 第一章 引言

### 1.1 研究背景

铁路轨道作为列车运行的基础设施，其几何状态直接影响列车的安全性和舒适性。轨道高低不平顺是指轨道实际轨面相对于理想平顺基准线的垂向偏差，是轨道几何状态评价的核心指标之一。长期存在的高低不平顺会导致轮轨作用力增大、轨道结构加速疲劳，是引起列车振动、晃车的直接原因，严重时甚至危及行车安全。

随着我国铁路运营里程的持续增长和列车运行速度的不断提升，轨道线路的养护维修工作量急剧增加。传统人工巡检方式效率低、主观性强、无法实现连续跟踪检测，已难以满足现代化铁路运营的需求。工务部门迫切需要高效、精准、低成本的新型轨道检测技术，以实现从"故障修"到"状态修"的管理模式转变。

轨道高低不平顺检测的核心挑战在于：**如何在高行驶速度下获取连续、高精度、左右轨独立的轨面高程序列**。这要求检测系统同时满足以下条件：（1）测量精度达到毫米级；（2）具备左右轨信号分离能力；（3）能够在复杂现场环境下稳定工作；（4）具备实时输出能力和数据后处理能力。满足上述条件的检测技术一直以来是铁路工程领域的研究重点。

##### （3）捷联式惯性测量方案的技术本质

无论日月明（FOG）还是瑞邦（MEMS），其核心测量链路均为**积分推算**：

```
陀螺仪 → ω(t)（角速率）→ 积分 → θ(t)（姿态角）
加速度计 → a(t)（加速度）→ 去除重力分量 → 积分 → v(t) → 积分 → 位移
倾角传感器 → θ_incli(t)（直接测角，无积分漂移）
                              ↓
                    多传感器融合处理
                              ↓
                    输出轨道几何参数
```

**捷联式方案的关键特征：**

| 特征 | 说明 |
|---|---|
| 倾角传感器的作用 | 直接测量俯仰角/横滚角，测量结果无积分漂移 |
| 陀螺仪的作用 | 积分姿态角，提供高频动态响应，弥补倾角传感器的慢动态缺陷 |
| 积分漂移的来源 | 主要来自陀螺仪角速率积分；倾角传感器本身不产生漂移 |
| FOG vs MEMS | FOG零偏稳定性约0.001°/h～0.01°/h；MEMS零偏稳定性约1°/h～10°/h，差距达2～3个数量级 |

因此，日月明的FOG方案精度高于瑞邦MEMS方案，根本原因在于FOG的陀螺仪积分漂移远小于MEMS。

**捷联式方案固有的技术局限：**

| 局限 | 说明 | 严重程度 |
|---|---|---|
| 积分漂移（陀螺仪）| FOG漂移虽小但仍累积；MEMS漂移显著 | 高（MEMS更严重）|
| 车体跳动干扰 | 悬挂系统振动与轨道不平顺频段重叠 | 中（可通过滤波抑制）|
| 悬挂耦合 | 车体悬挂可能吸收高频轨道病害，导致漏检 | 中 |
| 长波累积误差 | 18m基长内水平差分经多次积分，长距离后精度下降 | 中 |
| 无视觉检测 | 只能检测几何参数，无法检测轨面缺陷 | 高（功能盲区）|

#### 1.2.3 三角坑检测的行业标准方法

三角坑（Twist/Warp）是轨道几何状态的重要参数，定义为左右两股钢轨顶面在18米范围内出现三个及以上交替水平差的扭曲现象，表现为"正-负-正"或"负-正-负"状态。

**三点差分法（标准基长L=2.5m）：**

```
设里程s处超高值为u(s)，基长为L
三角坑值 = u(s+2L) - 2u(s+L) + u(s)
```

铁路标准限值：正线不大于6mm，其他线不大于8mm。三角坑超限时会导致转向架三点支承，低点车轮减载，严重时可能引发脱轨。

**三个方案均采用此标准三点差分法**，数据来源均为倾角传感器（或等效横滚角传感器）测得的超高差序列。本方案通过HWT905横滚角测量超高差，再进行三点差分计算三角坑值，原理上与日月明、瑞邦一致。

#### 1.2.4 激光轮廓测量技术与多传感器融合

3D激光轮廓仪是近年来在工业测量领域快速普及的传感器。其基于三角测量原理，向被测表面发射激光线，通过接收反射光斑在CMOS传感器上的位置精确解算被测点与传感器之间的距离：

$$D_s = \frac{f \cdot B}{p_s}$$

其中 $f$ 为聚焦透镜焦距，$B$ 为激光器与接收器基线距，$p_s$ 为光斑在CMOS上的像素偏移量。

3D激光轮廓仪具有以下优势：测量精度高（可达微米级）、非接触式测量、**无积分漂移**、响应速度快。将3D激光轮廓仪应用于轨道检测，可直接获取轨面三维廓形点云。

然而，3D激光单独使用时面临的问题是：测量结果受车体姿态影响较大。当车体发生横滚或俯仰时，激光测量的是"相对于倾斜车体的轨面位置"，而非"相对于绝对水平基准的轨道高程"。因此，激光需要与姿态传感器配合使用，才能获得真正的高不平顺数据——这正是本方案设计的出发点。

#### 1.2.5 本文技术路线与现有方案的根本差异

综合上述分析，本文提出的3D激光-惯性测量融合方案与日月明EBJ-3、瑞邦GTY-T-4的技术路线存在本质差异：

| 差异维度 | 捷联式惯性方案（日月明/瑞邦） | 本方案（3D激光+IMU融合） |
|---|---|---|
| 主测手段 | 惯性传感器（FOG/MEMS）积分推算 | **3D激光三角测量直接获取轨面位置** |
| 惯性传感器角色 | 核心测量单元 | 姿态校正辅助 |
| 积分漂移 | 存在（日月明小，瑞邦大） | **不存在（直接测量原理）** |
| 视觉检测能力 | 无 | **有（8大检测功能全覆盖）** |
| 传感器成本 | 高（FOG价格昂贵） | **低（HWT905 MEMS+工业激光）** |
| 技术路线本质 | 积分推算 | **直接测量+融合校正** |

本方案的创新性在于：从原理上摒弃了捷联式惯性方案的积分推算路线，改用3D激光进行直接测量，同时引入惯性传感器作为姿态校正辅助手段，实现了精度保障与成本控制的双重目标。



### 1.3 现有研究的核心问题与本文要解决的关键问题

基于上述分析，日月明EBJ-3和瑞邦GTY-T-4所代表的捷联式惯性测量方案虽然在1级～0级精度范围内能够工作，但其积分推算路线固有的漂移问题、视觉检测能力的缺失，以及FOG方案的高成本，始终是制约轨检仪进一步普及的技术瓶颈。现有激光+IMU融合研究尚缺乏系统性、完整性、可操作性的技术方案。综合而言，以下关键问题尚未得到充分解决：

**问题一：姿态耦合误差的精确量化与校正。** 激光与IMU安装位置的几何偏差会导致姿态角耦合进测量值，如何精确标定安装偏差参数并进行实时校正，是实现高精度检测的前提。

**问题二：车体跳动与轨道不平顺频谱分离。** 两者频段重叠，简单的频率滤波无法有效分离，需要从信号处理和滤波架构层面进行创新设计。

**问题三：融合参数的精确标定。** EKF的Q/R矩阵参数对滤波性能影响显著，但现有研究缺乏系统性的参数标定方法，特别是Q矩阵的过程噪声无法直接观测，需要引入Allan Variance等专用方法。

**问题四：现场条件下的持续精度保障。** 实验室标定参数在现场会随时间和温度发生漂变，如何建立可持续精化的校准体系，是工程落地的关键。

### 1.4 本文的主要工作与创新点

针对上述关键问题，本文的主要工作与创新点如下：

**创新点一：提出3D激光直接测量+IMU辅助校正的融合架构。** 以3D线激光为直接主测手段，以高精度IMU为姿态辅助校正手段，从原理上规避纯IMU方案的积分漂移和信号耦合问题。方案在物理上是可行的，在工程上是可实现的。

**创新点二：建立姿态耦合误差的精确几何校正模型。** 推导安装偏差与姿态耦合误差之间的定量关系，给出误差校正方程和参数标定方法，实现mm级残余误差控制。

**创新点三：设计三阶段校准体系。** 提出"出厂标定→启动自检→在线标定"三阶段校准体系，覆盖参数从建立到验证、从静态到动态、从一次性到持续精化的完整生命周期。

**创新点四：提出双速EKF架构，实现车跳干扰与轨道信号的频谱分离。** 设计快回路（200Hz）和慢回路（10Hz）分别处理高频姿态和低频轨廓，从滤波架构层面解决频段重叠问题。

**创新点五：给出完整的实车验证方案。** 设计精度对比、重复性、EKF收敛性、长时间稳定性四项验证实验，给出具体场地、里程范围、判定阈值和数据分析方法，可直接指导现场工程师执行。

### 2.1 需求分析

#### 2.1.1 检测对象

本系统针对铁路轨道高低不平顺进行检测。需要明确的是，本系统检测的是轨道相对不平顺值，而非轨道的绝对高程。绝对高程（如左轨距海平面的高度）对于轨道养护并无直接指导意义，而"某点相对于理想平顺轨面的偏差值"才是工务部门制定养护计划的依据。

轨道高低不平顺按波长可分为短波（λ<10m）和长波（λ≥10m）两类。短波不平顺主要影响乘车舒适度和轮轨作用力，长波不平顺主要影响轨道稳定性和列车蛇形运动。检测系统应能完整覆盖两类不平顺，并按波长分段输出评价指标。

#### 2.1.2 性能指标要求

根据铁路行业标准和实际养护需求，本系统的设计性能指标如下：

| 指标名称 | 设计指标 | 依据 |
|---|---|---|
| 检测速度 | 1 m/s（1000 mm/s）| 满足日常巡检效率需求 |
| 综合检测精度 | ≤ ±1mm | 铁路轨道不平顺检测行业标准 |
| 左右轨独立测量 | 必须支持 | 工务部门需要区分左右轨分别评价 |
| 连续检测里程 | 无限制 | 可实现不间断连续检测 |
| 数据输出延迟 | ≤ 1s | 满足实时告警需求 |
| TQI输出 | 支持 | 符合铁路行业数据格式规范 |
| 现场工作温度 | -20°C ~ +50°C | 适应铁路沿线环境 |
| 防护等级 | 传感器IP67，工控机IP54 | 适应现场粉尘、潮湿环境 |

### 2.2 系统总体架构

#### 2.2.1 检测机器人平台

本系统采用四轮轨道行驶平台作为检测机器人的运动载体。平台整体结构如下：

**行走机构**：四轮结构，左右轨各安装2个伺服电机驱动轮，四轮同步驱动，轮对轨距为铁路标准轨距1435mm。前后轮纵向轴距为1000mm，机器人整体呈对称布局。

**伺服驱动系统**：四个车轮分别由独立的伺服电机驱动，前进/后退速度可调，额定转速对应检测速度1m/s。伺服驱动通过EtherCAT总线与工控机通信，实现毫秒级响应控制。

**车体框架**：采用轻量化铝合金框架，结构强度满足传感器安装和现场作业要求。主体结构设置传感器安装基准面，确保各传感器相对位置的几何精度。

**3D激光安装位置**：左激光和右激光均安装在车体纵向中心线上，安装位置距前轴和后轴各500mm（即两轴正中间），激光垂直向下扫描对应轨面的轨头廓形。此布局确保激光测量点正好位于车体晃动最小的位置，姿态耦合误差最小。

**坐标系定义**：机器人坐标系定义如下——
- $X$轴：沿轨道纵向（行驶方向，前进为正）
- $Y$轴：沿轨道横向（垂直于轨道方向，左正右负）
- $Z$轴：垂直向上

```
        前端
    ←——— 1000mm ———→
    
  [左前轮]               [右前轮]
  伺服驱动                 伺服驱动
      \                   //
       \                 //
        \    [左激光]   [右激光] //
         \     ↓          ↓    //
          \  [车体中心]         //
           \                   //
         [左后轮]           [右后轮]
```

**悬挂系统**：前后轴各设置弹性悬挂节点，用于吸收轨道局部不平顺造成的冲击。悬挂系统固有频率设计在合理范围（约1~5Hz），与车体跳动频率对应。

**安全保护**：设置轨道限界保护装置，防止机器人意外掉道；设置急停按钮和远程停机功能，保障现场操作安全。

#### 2.2.3 传感器安装几何关系

3D激光传感器与HWT905姿态传感器在机器人坐标系中的相对位置是影响姿态耦合误差的关键参数。

设机器人坐标系定义如下：
- $X$轴：沿轨道纵向（行驶方向）
- $Y$轴：沿轨道横向（垂直于轨道方向）
- $Z$轴：垂直向上

**机器人坐标系定义：**
- $X$轴：沿轨道纵向（行驶方向，前进为正）
- $Y$轴：沿轨道横向（垂直于轨道方向，左正右负）
- $Z$轴：垂直向上

则各传感器在机器人坐标系中的安装位置如下：

| 传感器 | 安装位置（描述）| 坐标 $(X, Y, Z)$ | 数量 |
|---|---|---|---|
| 左侧3D激光 | 车体纵向中心线，距前轴500mm（两轴正中），垂直向下扫描左轨 | $(500, -\Delta x_L, -\Delta h_L)$ | ×1 |
| 右侧3D激光 | 车体纵向中心线，距前轴500mm（两轴正中），垂直向下扫描右轨 | $(500, +\Delta x_R, -\Delta h_R)$ | ×1 |
| HWT905 | 车体横向正中（Y=0），纵向与激光同位（X=500mm） | $(500, 0, 0)$ | ×1 |

其中，$\Delta x_L$、$\Delta x_R$ 为左右激光与HWT905之间的横向安装偏差（待标定），$\Delta h_L$、$\Delta h_R$ 为垂向安装偏差（待标定）。三个传感器共用同一纵向位置（X=500mm），简化了姿态耦合误差的标定关系。

#### 2.2.4 系统通信架构

系统采用三网物理隔离的通信架构，确保控制指令、采集数据和远程传输互不干扰：

**EtherCAT硬实时总线**：用于伺服电机驱动控制指令下发和状态反馈，周期时间≤1ms，满足毫秒级实时响应要求。

**千兆以太网**：专用于3D激光和工业相机的数据采集，带宽需求约为500Mbps（2台激光+6台相机同时工作），千兆以太网可满足带宽需求。

**4G/5G无线传输网**：用于检测数据远程上传、远程监控指令接收和状态数据上报。三网物理隔离可防止某一网络的故障影响其他网络的正常运行。

#### 2.2.5 供电系统

系统供电架构采用48V总线供电制式：

**主供电**：48V大容量锂电池组，容量满足连续运行4小时需求。

**智能电源管理（BMS）**：内置电池管理系统，实时监控电池组电压、电流、温度，具备过充、过放、过流、短路三重保护功能。

**供电分配**：
- 工控机：24V输入，IP54级隔离供电
- 激光传感器：24V供电，独立电源滤波
- 姿态传感器：12V供电，与工控机共地
- 伺服驱动：48V直供，EtherCAT通信隔离
