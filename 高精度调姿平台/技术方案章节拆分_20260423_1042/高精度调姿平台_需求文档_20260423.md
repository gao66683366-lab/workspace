# 高精度调姿平台需求文档

> 版本：V1.0
> 日期：2026年4月
> 用途：供方技术方案依据

---

## 一、项目名称与概述

项目名称：基于实时测距反馈的高精度调姿平台

项目概述：

设计并研制一套六自由度Stewart并联机构调姿平台，安装于运载小车上。平台在随小车行进过程中，通过底部单点测距传感器实时检测轨道高程，自动抵消轨道起伏，始终保持下平台水平。

---

## 二、平台基本参数

### 2.1 上下平台定义

| 名称 | 角色 | 说明 |
|---|---|---|
| 上平台 | 静平台 | 固定于运载小车，承受整体重量 |
| 下平台 | 动平台 | 由6组电动缸驱动，始终保持水平 |

### 2.2 尺寸与重量

| 参数 | 数值 | 说明 |
|---|---|---|
| 下平台直径 | φ200 mm | 铝合金，轻量化 |
| 上平台直径 | φ300 mm | 钢结构，承载基座 |
| 支链数量 | 6组 | 均匀分布，120°间隔 |
| 下平台总重量 | ≤ 2 kg | 含传感器支架，不含负载 |

---

## 三、调姿性能指标

### 3.1 核心性能指标

| 指标名称 | 目标值 | 说明 |
|---|---|---|
| 调姿精度 | ≤ ±0.05° | 横滚角和俯仰角均满足 |
| 全链路响应延迟 | ≤ 5 ms | 从测距采样到电动缸执行端到端 |
| 调姿响应带宽 | ≥ 2 Hz | 可补偿的轨道起伏频率上限 |
| 电动缸行程 | 20 mm | 紧凑设计 |
| 电动缸额定推力 | ≥ 200 N | 满足2kg负载及动态惯性力 |

### 3.2 精度说明

在双传感器间距 L = 300 mm 条件下：

- 单侧高差容限：0.26 mm（对应0.05°）
- 两侧最大允许高差：0.52 mm

## 四、执行机构要求

### 4.1 Stewart并联平台

| 参数 | 要求 |
|---|---|
| 机构类型 | 6-DOF Stewart并联机构 |
| 自由度 | 6个（X/Y/Z三轴平移 + 三轴旋转） |
| 电动缸类型 | 伺服电动缸（无杆式） |
| 电动缸行程 | 20 mm |
| 电动缸额定推力 | ≥ 200 N |
| 电动缸传动精度 | ±0.01 mm |
| 万向节类型 | 虎克铰/球铰混合 |

### 4.2 运动范围

| 参数 | 数值 |
|---|---|
| 最大横滚角 | ±15° |
| 最大俯仰角 | ±15° |
| 最大偏航角 | ±10° |
| 最大垂直位移 | ±10 mm |
| 重复定位精度 | ±0.02 mm |

## 五、控制系统要求

### 5.1 主控平台

| 参数 | 要求 |
|---|---|
| 主控型号 | Jetson AGX Orin |
| 算力 | 32 TOPS（INT8） |
| 用途 | 传感器数据处理、融合算法、控制算法运行 |

### 5.2 通信要求

| 参数 | 要求 |
|---|---|
| 实时总线 | EtherCAT |
| 控制周期 | 1 ms（1000 Hz） |
| 伺服驱动 | 6轴伺服驱动器，独立控制6个电动缸 |

### 5.3 控制链路延迟预算

| 环节 | 预算延迟 |
|---|---|
| 测距采样 | 0.5 ms |
| 数据处理与融合 | 2.0 ms |
| 逆运动学求解 | 0.5 ms |
| EtherCAT指令下发 | 0.5 ms |
| 伺服响应 | 1.0 ms |
| 电动缸机械响应 | 0.5 ms |
| 全链路总计 | ≤ 5 ms |

## 七、验收条件

### 7.1 静态调姿精度

| 验收项目 | 指标要求 | 测试方法 |
|---|---|---|
| 横滚角偏差 | ≤ ±0.05° | 静止状态下，平台调平后用角度传感器测量 |
| 俯仰角偏差 | ≤ ±0.05° | 静止状态下，平台调平后用角度传感器测量 |
| 重复性 | 连续10次调平，偏差≤±0.02° | 多次往复调平测试 |

### 7.2 动态调姿精度

| 验收项目 | 指标要求 | 测试方法 |
|---|---|---|
| 动态横滚角偏差 | ≤ ±0.1° | 车速1m/s，轨道起伏±5mm/0.5Hz条件下测量 |
| 动态俯仰角偏差 | ≤ ±0.1° | 车速1m/s，轨道起伏±5mm/0.5Hz条件下测量 |

### 7.3 响应延迟

| 验收项目 | 指标要求 | 测试方法 |
|---|---|---|
| 全链路延迟 | ≤ 5 ms | 从传感器信号触发到电动缸开始运动的示波器测量 |

### 7.4 外观与结构

| 验收项目 | 要求 |
|---|---|
| 外观 | 表面无划伤、变形，标识清晰 |
| 连接件 | 各连接部位牢固，无松动 |
| 防护 | 电动缸接口防护等级不低于IP65 |
